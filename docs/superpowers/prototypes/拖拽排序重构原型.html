当前场景：正常拖拽。长按任意卡片拖动，其他卡片平滑让位。

✅ 多选模式（参考 HomeViewModel）

  • ✓ 长按 0.5s + 抬手 → 进入多选（enterBatchMode）
  • ✓ 首个长按项自动选中（selectedIds = setOf(id)）
  • ✓ 点击其他项切换选中（toggleSelection）
  • ✓ 顶部栏：已选 X 项 + 全选/取消全选
  • ✓ 底部操作栏：置顶/完成/移动/复制/删除
  • ✓ 选中集合为空 → 自动退出多选
  • ✓ 点击 ✕ 或批量操作完成 → 退出多选
  • ⚠ 多选模式下禁用拖拽（isDragEnabled = !isBatchMode）

🎨 拖拽视觉方案

方案B：占位符 + 浮起跟随（已确认）
• 原位置保留虚线占位框（橙色虚线）
• 被拖卡片浮起 + 阴影 + 1.05x 缩放
• 卡片精确跟随手指位置
• 其他卡片平滑让位（弹性过渡）

实施提示：Calvin-LL/Reorderable 库的 ReorderableItem 内部会通过 graphicsLayer.translationY 移动整个 item。 虚线占位框需作为底层装饰渲染（如 Box 内底层 Canvas + 上层 ReorderableItem），避免跟随手指移动。
