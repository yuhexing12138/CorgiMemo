# 灵感卡片展示页手动测试清单

## 入口
- [ ] 灵感页时间线点击任意灵感 → 进入展示页
- [ ] 顶部 TopBar 高度符合 64dp 规范
- [ ] 返回箭头位置正确（左上）
- [ ] 右侧三个按钮（复制/编辑/分享）位置正确

## 内容渲染
- [ ] 标题字号、字重、颜色正确（16sp Medium，onSurface）
- [ ] 日期时间格式为 "yyyy.MM.dd HH:mm"
- [ ] 正文 lineHeight = 21sp
- [ ] 图片缩略图尺寸 120dp，圆角 6dp
- [ ] 标签背景 #FFF3E0，文字 UiColors.Primary
- [ ] 字数徽章显示在右上角，格式 "N字"
- [ ] Logo 居中显示 "- 简记事 -"

## 左右滑动
- [ ] HorizontalPager 切换顺畅
- [ ] 切换时三个功能按钮回调更新为当前灵感
- [ ] 第一篇左滑有回弹
- [ ] 最后一篇右滑有回弹
- [ ] 切换后字数徽章、标题、正文同步更新

## 复制功能
- [ ] 点击复制按钮 → Snackbar 显示 "文本内容已复制到剪切板"
- [ ] Snackbar 2 秒后自动消失
- [ ] 粘贴到记事本，验证格式：
  - 标题换行
  - 日期换行
  - 空行
  - 正文换行
  - 空行
  - 标签（无标签时无标签行）

## 编辑功能
- [ ] 点击编辑按钮 → 进入编辑页
- [ ] 编辑页内容与展示页一致
- [ ] 从编辑页返回 → 展示页数据自动刷新

## 分享功能
- [ ] 点击分享按钮 → 弹出底部菜单
- [ ] 弹窗显示三个选项：保存到相册 / 更多分享 / 取消
- [ ] 点击"保存到相册" → Snackbar 显示"已保存到相册：Inspiration_xxx.png"
- [ ] 在文件管理器或相册中查看保存的 PNG 文件
- [ ] 点击"更多分享" → 弹出系统分享面板
- [ ] 选择微信/QQ → 能正常分享图片
- [ ] 截图内容包含：标题、日期、正文、图片、标签、Logo（不包含 TopBar 和字数徽章）

## 图片预览
- [ ] 点击卡片中的图片 → 全屏打开
- [ ] 全屏背景为黑色
- [ ] 多图可左右滑动翻页
- [ ] 顶部显示页码 "1/N"
- [ ] 双指捏合可缩放（1x~4x）
- [ ] 双击可放大/还原
- [ ] 点击右上角 X 按钮可关闭
- [ ] 点击系统返回键可关闭

## 响应式
- [ ] 平板（≥600dp）：卡片最大宽度 480dp 居中（待后续优化）
- [ ] 手机（<600dp）：卡片充满内容区

## 错误处理
- [ ] 不存在的灵感 ID：显示加载圈
- [ ] 图片文件丢失：图片占位（不崩溃）
- [ ] 截图失败：Snackbar 提示"分享失败，请重试"

## 性能
- [ ] 进入展示页首次渲染 < 200ms
- [ ] 左右滑动切换无明显卡顿
- [ ] 截图保存到相册 < 1s
