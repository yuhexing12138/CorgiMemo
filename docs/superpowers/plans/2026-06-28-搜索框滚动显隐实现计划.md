# 搜索框滚动显隐 实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 实现待办页面搜索框的滚动显隐交互——手指向下滑时滑入显示，手指向上滑时滑出隐藏，到顶部时隐藏，输入中和搜索为空时始终显示。

**Architecture:** 在 ReorderableLazyColumn 中暴露 LazyListState 参数，HomeScreen 使用 snapshotFlow 检测滚动方向，用 AnimatedVisibility + slideInVertically/slideOutVertically 实现搜索框的滑入滑出动画。

**Tech Stack:** Jetpack Compose (AnimatedVisibility, slideInVertically/slideOutVertically, derivedStateOf, snapshotFlow, LazyListState)

- [ ] **Step 3: 添加滚动方向检测逻辑**

在 `val searchQuery by viewModel.searchQuery.collectAsState()` 之后（约第140行区域），添加滚动方向检测：

```kotlin
/** 滚动方向检测：手指向下滑 = 列表往上走 = isScrollingDown */
var prevIndex by remember { mutableIntStateOf(0) }
var prevOffset by remember { mutableIntStateOf(0) }
var isScrollingDown by remember { mutableStateOf(false) }

LaunchedEffect(lazyListState) {
    snapshotFlow {
        lazyListState.firstVisibleItemIndex to lazyListState.firstVisibleItemScrollOffset
    }.distinctUntilChanged().collect { (currentIndex, currentOffset) ->
        isScrollingDown = if (currentIndex != prevIndex) {
            currentIndex < prevIndex
        } else {
            currentOffset < prevOffset
        }
        prevIndex = currentIndex
        prevOffset = currentOffset
    }
}

val isAtTop by remember {
    derivedStateOf {
        lazyListState.firstVisibleItemIndex == 0 &&
            lazyListState.firstVisibleItemScrollOffset == 0
    }
}

/** 搜索框可见性：向下滚动且不在顶部 + 有搜索词时始终显示 */
val searchBarVisible by remember {
    derivedStateOf {
        (isScrollingDown && !isAtTop) || searchQuery.isNotBlank()
    }
}
```
