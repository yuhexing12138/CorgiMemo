# 野兽派柯基流场粒子系统设计

## 概述

将 `corgimemo-showcase.html` 中的全局背景飘动粒子从 CSS 动画重构为 p5.js 驱动的流场粒子系统，使用柯基 PNG 图片作为粒子形态，融入野兽派视觉风格。

## 设计决策

| 决策项 | 选择 | 原因 |
|--------|------|------|
| 实现方案 | p5.js 全屏 Canvas 覆盖层 | 性能最优，野兽派效果自由度最高 |
| 野兽派程度 | 适度 | 与页面"杂志风85%+野兽派15%"风格协调 |
| 粒子形态 | 柯基 PNG 图片 | 用户需求，增强品牌识别 |
| 运动方式 | Perlin 噪声流场 | 有机飘动，与野兽派硬边形成张力 |
| 卡顿效果 | 不使用 | 用户明确要求粒子运动流畅 |

## 技术架构

```
┌─────────────────────────────────────┐
│  页面内容 (z-index: 10+)            │
├─────────────────────────────────────┤
│  p5.js Canvas (z-index: 0)          │
│  - fixed 全屏定位                    │
│  - pointer-events: none             │
│  - 透明背景，仅绘制粒子              │
├─────────────────────────────────────┤
│  页面背景 (CSS --color-bg-paper)     │
└─────────────────────────────────────┘
```

## 粒子系统规格

| 属性 | 值 | 说明 |
|------|-----|------|
| 粒子形态 | 柯基 PNG 图片 | 随机选取 11 种姿态的第一帧 |
| 粒子大小 | 20-50px | 大小差异明显 |
| 粒子数量 | 15-25 个 | 图片粒子比符号重，不宜过多 |
| 运动方式 | Perlin 噪声流场 | 有机飘动 |
| 边框 | 2-3px 粗黑边框 | 野兽派特征 |
| 阴影 | 硬偏移阴影 4-6px | 无模糊，野兽派特征 |
| 配色 | 野兽派色板随机 | 橙红/暖黄/薄荷绿/深黑 |
| 透明度 | 0.15-0.35 | 背景装饰级别 |
| 旋转 | 跟随流场，量化到 45° 倍数 | 注入网格感 |

## 流场参数

| 参数 | 默认值 | 范围 | 说明 |
|------|--------|------|------|
| noiseScale | 0.003 | 0.001-0.01 | 噪声缩放 |
| flowSpeed | 0.8 | 0.2-2.0 | 流动速度 |
| particleCount | 20 | 10-30 | 柯基粒子数量 |
| brutalBorder | 2 | 1-4 | 边框粗细 |
| brutalShadow | 5 | 2-10 | 硬阴影偏移 |
| opacity | 0.25 | 0.1-0.5 | 粒子透明度 |

## 柯基图片资源

路径：`../app/src/main/res/drawable/corgi_*_01.png`（每种姿态取第一帧）

可用姿态：tilt, stand, sleep, sit, shy, sad, run, roll, proud, lie, wag
共 11 张图片

## 代码变更清单

### 删除

### 新增

### 保留

- 其他所有页面结构和功能不变

## 野兽派视觉特征

1. **粗边框**：每个柯基粒子绘制 2-3px 黑色描边
2. **硬阴影**：纯色偏移阴影，无高斯模糊
3. **旋转量化**：角度量化到 45° 倍数，注入网格感
4. **色块叠加**：粒子背后绘制野兽派色板色块
5. **大小对比**：粒子大小差异明显，不追求均匀
