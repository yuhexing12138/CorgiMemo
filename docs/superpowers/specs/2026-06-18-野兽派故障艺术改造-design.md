## 一、设计目标

对 `corgimemo-showcase.html` 进行三项系统性改造:

1. **Tab1-6 野兽派主导改造**: 参照参考文件设计风格,将 tab1-tab6 主题组件全面改造为鲜明的野兽派艺术风格,以暖色调为主,与现有杂志编辑风格形成强烈视觉对比。
2. **Tab7 故障艺术改造**: 将 tab7 改造为故障艺术风格,融合柯基暖色基调、故障艺术视觉元素及杂志编辑风格,外壳故障+内容暖色。
3. **内容重组**: 识别相似度高的标题与内容区块,跨 Tab 合并重组,提升信息传达效率。

### 核心约束

- **文字可见性**: 所有文字必须在视觉上清晰可见,不得藏于背景色中。色块背景必须搭配高对比度文字。
- **响应式兼容**: 桌面/平板/移动端均保持风格一致性与功能完整性。
- **函数级注释**: 代码变更需添加中文函数级注释与关键逻辑注释。

## 二、野兽派设计语言规范(Tab1-6 主导风格)

### 2.1 设计原则

野兽派作为 tab1-6 的**主导视觉风格**(非点缀),杂志风退为布局骨架。重点突出野兽派的粗犷线条、大胆色彩运用和原始质感特征。

### 2.2 视觉要素对比

| 设计要素 | 杂志风(退为骨架) | 野兽派(主导视觉) |
|---------|-----------------|-----------------|
| 边框 | 2px 细线 | 3-4px 粗黑边框 `#1a1a1a` |
| 阴影 | 4px 软阴影 | 6-8px 硬阴影无模糊 `5px 5px 0` |
| 圆角 | 20px 大圆角 | 0-4px 极小圆角(原始质感) |
| 色彩 | 米白底+橙红点缀 | 大胆色块:橙红/橙黄/薄荷/暖黄交替 |
| 字体 | 衬线优雅 | 粗体无衬线 font-weight:900,超大字号 |
| 布局 | 对称网格 | 错落不等高、不对称分栏 |

### 2.3 暖色调色板

| 色值 | 用途 | 文字搭配 |
|------|------|---------|
| `#FF6B35` | 主色橙红(柯基色) | 白色 `#fff` 文字 |
| `#FFD4A8` | 暖橙黄 | 深黑 `#1a1a1a` 文字 |
| `#FFB74D` | 暖黄 | 深黑 `#1a1a1a` 文字 |
| `#E8743C` | 深橙 | 白色 `#fff` 文字 |
| `#7EC8A0` | 薄荷绿(辅助对比) | 深黑 `#1a1a1a` 文字 |
| `#1a1a1a` | 深黑(边框/文字) | — |
| `#f5f0e8` | 米白主背景 | 深黑文字 |

### 2.4 组件改造清单

#### 章节号 `.chapter-number`
- 字号: 80-96px(桌面) / 56px(平板) / 48px(移动)
- font-weight: 900
- 粗黑边框包裹(3px solid #1a1a1a)
- 硬阴影色块(6px 6px 0)
- 暖色填充交替(橙红/薄荷/暖黄)

#### 卡片 `.brutal-card`
- 边框: 3px solid #1a1a1a
- 圆角: 0-2px(原始质感)
- 硬阴影: 6px 6px 0(无模糊)
- 背景色块交替: #fff / #FF6B35 / #FFD4A8 / #7EC8A0
- hover: 阴影增至 8px 8px 0,位移 -2px,-2px

#### 标签 `.diff-tag`
- 粗框(2px solid #1a1a1a)
- 硬阴影(3px 3px 0)
- 色块背景
- font-weight: 900

#### 网格 `.brutal-grid`
- 错落不等高布局
- 部分卡片 margin-top 错位(8-16px)
- 非等高排列

#### 引言 `.editorial-quote`
- 粗框包裹(3px solid #1a1a1a 上下)
- 暖色块背景(#FFD4A8 或 #FF6B35)
- 高对比度文字

### 2.5 文字可见性保障规则

- `#FF6B35` 橙红背景 → 白色 `#fff` 文字
- `#FFD4A8` 浅橙背景 → 深黑 `#1a1a1a` 文字
- `#FFB74D` 暖黄背景 → 深黑 `#1a1a1a` 文字
- `#E8743C` 深橙背景 → 白色 `#fff` 文字
- `#7EC8A0` 薄荷背景 → 深黑 `#1a1a1a` 文字
- `#1a1a1a` 深黑背景 → 白色 `#fff` 或 `#f5f0e8` 文字
- `#f5f0e8` 米白背景 → 深黑 `#1a1a1a` 文字

## 三、Tab7 故障艺术外壳 + 暖色内容

### 3.1 三层结构

| 层级 | 风格 | 元素 |
|-----|------|------|
| 外壳层 | 故障艺术 | 手机模拟器边框(RGB分离霓虹)、章节号(glitch抖动)、操作提示(终端等宽)、扫描线动画 |
| 内容层 | 柯基暖色 | 手机屏幕内部UI保留 #FF6B35/#f5f0e8 暖色基调,屏幕内卡片用暖色野兽派 |
| 过渡层 | 杂志风 | 章节标签、引言保持杂志骨架 |

### 3.2 故障艺术色彩(外壳)

| 色值 | 用途 |
|------|------|
| `#00ff9f` | 霓虹绿(主故障色) |
| `#ff006e` | 霓虹粉(RGB分离) |
| `#0095ff` | 电光蓝(RGB分离) |
| `#0a0a0f` | 故障深背景 |

### 3.3 故障效果

- **RGB 分离文字**: `text-shadow: 2px 0 #ff006e, -2px 0 #0095ff`
- **扫描线动画**: `linear-gradient` 4px 条纹 + `scanline` 动画
- **glitch 抖动**: `glitchShift` 关键帧动画(3s 循环)
- **终端式提示**: 等宽字体 `var(--font-mono)`,霓虹绿文字

### 3.4 暖色内容层

- 手机屏幕内部 UI 保留柯基暖色基调
- 屏幕内卡片/按钮采用暖色野兽派风格(粗框+硬阴影+暖色块)
- 屏幕内文字确保高对比度可见性
- 形成冷色故障外壳 + 暖色内容的视觉对比
