#### 11.4.2 天数卡片详细设计

| 属性 | 值 | 规范引用 |
|------|-----|---------|
| **尺寸** | 宽度约屏幕1/2（网格布局，2列） | 自适应布局 |
| **圆角** | 16dp | 12.1.5.1 按钮圆角 |
| **背景** | #FFFFFF（亮色）/ #2A2A2A（深色） | 12.1.2.2 卡片背景 |
| **阴影** | elevation 2dp | 12.1.6 卡片默认阴影 |
| **标题** | 14sp Medium，最多1行 | 12.1.4.2 辅助文字 |
| **天数** | 24sp Bold | 倒计时主色#FF9A5C，正计时成功色#7EC8A0，已过期#999999 |
| **日期文字** | "还有"/"已经"/"已过" + "X天" | 状态描述 |
| **目标日期** | 12sp Regular，#999999 | 12.1.2.2 提示文字，格式"YYYY年M月D日" |
| **缩略图** | 32×32dp，圆角8dp | 最多显示2张 |
| **关联提示** | 12sp Regular，#666666 | 12.1.2.2 次要文字 |

**颜色规范引用**（见12.1.2.3功能色）：
- 倒计时：主色 #FF9A5C（温暖、期待）
- 正计时：成功色 #7EC8A0（纪念、成就）
- 已过期：提示文字 #999999（淡化）

#### 11.4.3 天数卡片颜色规则

| 状态 | 天数颜色 | 标题颜色 | 背景变化 | 设计意图 |
|------|---------|---------|---------|---------|
| **即将到来（≤3天）** | #FF8A80 高优先级 | #FF8A80 | 左侧4dp高优先级色竖条 | 紧急提醒，但柔和不焦虑 |
| **即将到来（4-30天）** | #FF9A5C 主色 | 主文字 #2D2D2D | 无 | 正常期待感 |
| **即将到来（>30天）** | #999999 提示文字 | 主文字 #2D2D2D | 无 | 淡化显示 |
| **正在纪念** | #7EC8A0 成功色 | 主文字 #2D2D2D | 无 | 纪念、成就感 |
| **已过期** | #999999 提示文字 | #999999 | 整体透明度0.6 | 淡化处理 |

**设计规范引用**：
- 颜色：遵循12.1.2.3功能色规范（高优先级#FF8A80、成功色#7EC8A0）
- 文字：遵循12.1.2.2中性色规范（主文字#2D2D2D、提示文字#999999）
- 强调：≤3天使用高优先级色（柔和红色），避免传统警告红的焦虑感

#### 11.4.7 检查点（特殊日期部分）

- [ ] 日期列表分三组：即将到来、正在纪念、已过期
- [ ] 天数卡片显示大字天数+目标日期
- [ ] 天数颜色规则正确（≤3天红色、正计时绿色等）

#### 11.4.8 真机验证步骤（特殊日期部分）

1. 点击📅日期Tab → 应进入日期列表页
2. 日期列表应分三组：即将到来、正在纪念、已过期
3. 点击中央⊕ → 点击"📅日期"气泡 → 应进入日期编辑页
4. 输入标题、选择日期 → 保存 → 应返回列表并显示天数卡片
5. 天数应正确计算（倒计时/正计时）
6. ≤3天的卡片天数应为红色
7. 正计时卡片天数应为绿色
