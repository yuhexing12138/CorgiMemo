#### 11.4.7 检查点（特殊日期部分）

- [ ] 日期列表分三组：即将到来、正在纪念、已过期
- [ ] 天数卡片显示大字天数+目标日期
- [ ] 天数颜色规则正确（≤3天红色、正计时绿色等）
- [ ] 日期编辑页支持日期选择器和计时模式切换
- [ ] 日期编辑页支持插入图片
- [ ] 日期编辑页支持关联其他卡片
- [ ] 日期空状态显示引导文案+CTA按钮

#### 11.4.8 真机验证步骤（特殊日期部分）

1. 点击📅日期Tab → 应进入日期列表页
2. 日期列表应分三组：即将到来、正在纪念、已过期
3. 点击中央⊕ → 点击"📅日期"气泡 → 应进入日期编辑页
4. 输入标题、选择日期 → 保存 → 应返回列表并显示天数卡片
5. 天数应正确计算（倒计时/正计时）
6. ≤3天的卡片天数应为红色
7. 正计时卡片天数应为绿色
