# 工具栏统一设计

> **日期**: 2026-07-10
> **范围**: 灵感编辑页底部工具栏（上层 RichTextFormatToolbar + 下层 InspirationEditBottomBar）
> **目标**: 统一上下两层工具栏的高度、背景色、按钮尺寸和图标样式

## 背景

灵感编辑页底部工具栏由两层组成：
- **上层**：`RichTextFormatToolbar` — 点击下层"⋮"按钮后展开的富文本格式工具栏（B/I/U/S、列表、对齐、链接、代码）
- **下层**：`InspirationEditBottomBar` 底部按钮行 — 始终显示的 5 个核心按钮（相机、语音、标签、@、⋮）

当前两层在高度、背景色、按钮尺寸、内容元素上存在显著差异，需要统一以提升视觉一致性。

## 改动前状态

| 属性 | 上层 (RichTextFormatToolbar) | 下层 (BottomBar 底部行) |
|------|------|------|
| 容器 | `Row` (无 Surface) | `Surface` (圆角12dp + 阴影4dp) |
| 背景色 | `colorScheme.surface` (白色) | `colorScheme.background` (暖米色 #FFFBF5) |
| 垂直内边距 | 8dp | 10dp |
| 水平内边距 | 12dp | 16dp |
| 按钮容器 | `Box.size(36.dp)` | `IconButton.size(40.dp)` |
| 内容元素 | `Text` (文字 B/I/U/S/•≡/1./⬅/⬌/➡/🔗/</>)  | `Icon` (Material 图标) |
| 内容大小 | 15sp / 16sp / 18sp (字号) | 22dp (图标尺寸) |
| 激活背景色 | `#FFE0C0` (浅暖橙) | 无 |
| 激活内容色 | `#E88A4D` (深暖橙) | `#FF9A5C` (primary) |
| 估算高度 | ~52dp | ~60dp |

## 设计决策

### 1. 高度统一 → 56dp (Material 标准)

- **统一高度**: 56dp (Material Design 官方推荐工具栏高度)
- **计算方式**: 垂直内边距 8dp × 2 + IconButton 40dp = 56dp
- **上层**: 保持垂直内边距 8dp，按钮从 Box 36dp → IconButton 40dp
- **下层**: 垂直内边距从 10dp → 8dp，按钮保持 IconButton 40dp

### 2. 背景色统一 → 暖米色 (colorScheme.background)

- **上层**: 移除 `Row.background(colorScheme.surface)`，改为继承父容器 `Surface` 的暖米色背景
- **下层**: 不变 (已是 `colorScheme.background`)
- **原理**: 上层格式工具栏在 `InspirationEditBottomBar` 的 `Surface` 内部，移除自身背景后自动显示父容器的暖米色

### 3. 按钮组件统一 → IconButton 40dp + Icon 22dp

- **上层**: 将 `FormatToolButton` (Box 36dp + Text) 替换为 `FormatIconButton` (IconButton 40dp + Icon 22dp)
- **下层**: 不变 (已使用 IconButton 40dp + Icon 22dp)
- **组件签名**: `FormatIconButton(imageVector: ImageVector, isActive: Boolean, onClick: () -> Unit, contentDescription: String)`

### 4. 图标映射

| 功能 | 当前 (Text) | 改动后 (Material Icon) | 图标描述 |
|------|------|------|------|
| 加粗 | B | `FormatBold` | 字母 B 加粗 |
| 斜体 | I | `FormatItalic` | 字母 I 倾斜 |
| 下划线 | U | `FormatUnderlined` | 字母 U 带下划线 |
| 删除线 | S | `FormatStrikethrough` | 字母 S 带删除线 |
| 无序列表 | •≡ | `FormatListBulleted` | 三横线，每线前一个圆点 |
| 有序列表 | 1. | `FormatListNumbered` | 三横线，每线前数字 1/2/3 |
| 左对齐 | ⬅ | `FormatAlignLeft` | 四横线(长-短-长-短)，左对齐 |
| 居中对齐 | ⬌ | `FormatAlignCenter` | 四横线(长-短-长-短)，居中对齐 |
| 右对齐 | ➡ | `FormatAlignRight` | 四横线(长-短-长-短)，右对齐 |
| 插入链接 | 🔗 | `Link` | 链条图标 |
| 代码块 | </> | `Code` | 尖括号 `</>` 图标 |

所有图标来自 `androidx.compose.material.icons.filled`，无需额外依赖。

### 5. 激活状态统一

| 属性 | 改动前 | 改动后 |
|------|--------|--------|
| 激活背景色 | `#FFE0C0` (浅暖橙) | 保留 `#FFE0C0` |
| 激活图标色 | `#E88A4D` (深暖橙) | `#FF9A5C` (主题 primary，与下层 ⋮ 按钮激活色一致) |
| 非激活图标色 | `onSurfaceVariant` | 不变 |

### 6. 分隔符

- 保留 4 组之间的竖线分隔符 (`ToolbarDivider`)
- 高度从 24dp 调整为 28dp (适配更大的 40dp 按钮)

## 不变项

- 下层 5 个按钮内容不变 (PhotoCamera, Mic, Tag, AlternateEmail, MoreVert)
- 下层 `Surface` 的圆角(12dp)和阴影(4dp)不变
- 上层 `horizontalScroll` 可滚动不变
- `AnimatedVisibility` 展开/折叠动画不变
- 上层 `FormatButtonGroup` 分组结构和 `Arrangement.spacedBy(2.dp)` 不变

## 涉及文件

| InspirationEditBottomBar.kt | `app/src/main/java/com/corgimemo/app/ui/screens/inspiration/components/InspirationEditBottomBar.kt` | 垂直内边距 10dp → 8dp |

## 验证标准

1. 上下两层工具栏视觉高度一致 (56dp)
2. 上下两层背景色一致 (暖米色 #FFFBF5)
3. 上下两层按钮大小一致 (IconButton 40dp + Icon 22dp)
4. 上层所有按钮使用 Material Icon (无 Text 文字按钮)
5. 激活状态颜色与主题 primary (#FF9A5C) 一致
6. 编译通过，无 import 缺失
