核心 API：LazyListLayoutInfo

❌ 当前限制

→ 只能保存 1 次操作，新操作覆盖旧操作

✅ 改进方案：Stack 结构

// 数据结构变更
private val _reorderUndoStack = MutableStateFlow(
    ArrayDeque<ReorderUndoData>()  // 支持多级
)
val maxUndoStackSize = 10  // 上限 10 次

// 操作：推入栈顶
_reorderUndoStack.value.addFirst(newData)
if (size > maxUndoStackSize) removeLast()

💡 设计决策：上限设为 10 次足够日常使用， 避免内存无限增长。每次 undo 后重新 loadTodos() 确保一致性。
