# 灵感页三点菜单弹窗设计规范

> **日期:** 2026-07-10
> **状态:** 已确认
> **方案:** 方案 A — 独立实现（不干扰待办页代码）

---

## 1. 目标

为灵感页右上角三点图标添加功能菜单弹窗，参考待办页的 `TodoMenuDropdown` 模式。弹窗包含 7 个菜单项，其中 2 项实际开发（隐藏详情、批量选择），5 项占位显示"开发中"。

批量选择模式支持删除和置顶 2 个实际操作，其余 3 个（分享、创建副本、转换为待办）占位。

## 5. 批量选择模式 — ViewModel

### InspirationViewModel 新增状态

```kotlin
// 菜单弹窗展开状态
private val _menuExpanded = MutableStateFlow(false)
val menuExpanded: StateFlow<Boolean> = _menuExpanded.asStateFlow()

// 批量模式状态
private val _isBatchMode = MutableStateFlow(false)
val isBatchMode: StateFlow<Boolean> = _isBatchMode.asStateFlow()

private val _selectedInspirationIds = MutableStateFlow<Set<Long>>(emptySet())
val selectedInspirationIds: StateFlow<Set<Long>> = _selectedInspirationIds.asStateFlow()
```

### InspirationViewModel 新增方法

```kotlin
fun setMenuExpanded(expanded: Boolean) {
    _menuExpanded.value = expanded
}

fun enterBatchMode() {
    _isBatchMode.value = true
    _selectedInspirationIds.value = emptySet()
}

fun exitBatchMode() {
    _isBatchMode.value = false
    _selectedInspirationIds.value = emptySet()
}

fun toggleSelection(inspirationId: Long) {
    _selectedInspirationIds.value = _selectedInspirationIds.value.let {
        if (it.contains(inspirationId)) it - inspirationId else it + inspirationId
    }
}

fun selectAllInspirations() {
    _selectedInspirationIds.value = _inspirations.value.map { it.id }.toSet()
}

fun clearSelection() {
    _selectedInspirationIds.value = emptySet()
}

fun batchDeleteInspirations() {
    val selectedIds = _selectedInspirationIds.value
    if (selectedIds.isEmpty()) return
    viewModelScope.launch {
        selectedIds.forEach { id ->
            inspirationRepository.deleteInspiration(id)
        }
        exitBatchMode()
    }
}

fun batchPinInspirations() {
    val selectedIds = _selectedInspirationIds.value
    if (selectedIds.isEmpty()) return
    viewModelScope.launch {
        val inspirations = _inspirations.value.filter { it.id in selectedIds }
        val allPinned = inspirations.all { it.isPinned }
        inspirations.forEach { insp ->
            val updated = insp.copy(isPinned = !allPinned)
            inspirationRepository.updateInspiration(updated)
        }
        exitBatchMode()
    }
}
```

## 6. 时间线节点视觉变化

### TimelineInspirationItem 新增参数

```kotlin
isBatchMode: Boolean = false,
isSelected: Boolean = false
```

### 节点视觉规格

| 模式 | 节点视觉 | 尺寸 | 颜色 |
|------|----------|------|------|
| 普通模式 | 实心圆点 | 直径 6dp | 置顶=`Color(0xFFFF9A5C)`，普通=`MaterialTheme.colorScheme.primary` |
| 批量未选中 | 空心圆圈（边框 2dp） | 直径 16dp | 边框=`Color(0xFFCCCCCC)` |
| 批量已选中 | 实心圆 + 白色勾号√ | 直径 16dp | 背景=`UiColors.Primary`，勾号=`Color.White` |

### 实现方式

```kotlin
val nodeSize = if (isBatchMode) 16.dp else 6.dp
val nodeCenterY = if (isBatchMode) 18.dp else 11.dp  // 批量模式节点更大，Y 位置微调对齐标题中心

Box(
    modifier = Modifier
        .offset(x = nodeCenterX - nodeSize / 2, y = nodeCenterY - nodeSize / 2)
        .size(nodeSize)
        .then(
            if (isBatchMode) {
                if (isSelected) {
                    Modifier
                        .background(UiColors.Primary, CircleShape)
                } else {
                    Modifier
                        .border(2.dp, Color(0xFFCCCCCC), CircleShape)
                }
            } else {
                Modifier.background(nodeColor, CircleShape)
            }
        )
        .align(Alignment.TopStart)
) {
    if (isBatchMode && isSelected) {
        Icon(
            imageVector = Icons.Default.Check,
            contentDescription = "已选中",
            tint = Color.White,
            modifier = Modifier.size(12.dp).align(Alignment.Center)
        )
    }
}
```

### 点击行为切换

- 普通模式：`onClick` → 跳转编辑页 `navController.navigate("inspiration_edit/${inspiration.id}")`
- 批量模式：`onClick` → `viewModel.toggleSelection(inspiration.id)`
- 批量模式：`onLongClick` 禁用（不触发长按操作面板）

## 7. InspirationBatchActionBar 设计

### 组件签名

```kotlin
@Composable
fun InspirationBatchActionBar(
    isBatchMode: Boolean,
    selectedInspirationIds: Set<Long>,
    totalInspirationCount: Int,
    onSelectAll: () -> Unit,
    onClearSelection: () -> Unit,
    onDelete: () -> Unit,
    onPin: () -> Unit
)
```

### 布局

```
[全选/取消全选]                          [分享] [删除] [置顶] [副本] [转待办]
```

- 左侧：`TextButton` 全选/取消全选（文字随状态切换）
- 右侧：5 个 `IconButton`（40dp），水平排列，`spacedBy(8.dp)`

### 图标按钮列表

| # | 功能 | 图标 | 状态 | enabled | 视觉 |
|---|------|------|------|---------|------|
| 1 | 分享 | `Icons.Outlined.Share` | 🔒 占位 | `false` | `alpha(0.4f)` |
| 2 | 删除 | `Icons.Outlined.Delete` | ✅ 开发 | `hasSelection` | 正常/`alpha(0.38f)` |
| 3 | 置顶 | `Icons.Outlined.PushPin` | ✅ 开发 | `hasSelection` | 正常/`alpha(0.38f)` |
| 4 | 创建副本 | `Icons.Outlined.ContentCopy` | 🔒 占位 | `false` | `alpha(0.4f)` |
| 5 | 转换为待办 | `Icons.Outlined.SwapHoriz` | 🔒 占位 | `false` | `alpha(0.4f)` |

### 动画

使用 `AnimatedVisibility` + `slideInVertically`/`slideOutVertically`，与 `HomeBatchActionBar` 一致。

### 删除确认弹窗

在 `InspirationScreen` 中渲染 `AlertDialog`：
- 标题："删除选中项"
- 内容："确定要删除已选择的 {N} 条灵感吗？\n此操作不可撤销。"
- 确认按钮：红色"删除" → `viewModel.batchDeleteInspirations()`
- 取消按钮："取消"

## 8. MainScreen 集成

### 状态收集

```kotlin
val inspirationMenuExpanded by inspirationViewModel.menuExpanded.collectAsState()
val inspirationIsBatchMode by inspirationViewModel.isBatchMode.collectAsState()
val inspirationSelectedIds by inspirationViewModel.selectedInspirationIds.collectAsState()
```

### 有效批量模式判断

```kotlin
val effectiveBatchMode = when (selectedTab) {
    TabItem.TODO -> isBatchMode                    // HomeViewModel
    TabItem.INSPIRE -> inspirationIsBatchMode      // InspirationViewModel
    else -> false
}

val effectiveSelectedCount = when (selectedTab) {
    TabItem.TODO -> selectedTodoIds.size
    TabItem.INSPIRE -> inspirationSelectedIds.size
    else -> 0
}
```

### topBar 改动

1. **onMoreClick**：灵感页从 `Toast` 改为 `inspirationViewModel.setMenuExpanded(true)`
2. **dropdownContent**：灵感页渲染 `InspirationMenuDropdown`
3. **标题**：批量模式时显示"选中 {effectiveSelectedCount} 项"
4. **actionButtons**：批量模式时清空
5. **centerContent**：灵感页批量模式时不显示日期点击
6. **左侧返回箭头**：批量模式时显示，点击退出批量模式
   - TODO 页：`homeViewModel.exitBatchMode()`
   - INSPIRE 页：`inspirationViewModel.exitBatchMode()`

### bottomBar 槽位切换

```kotlin
bottomBar = {
    when {
        effectiveBatchMode && selectedTab == TabItem.TODO ->
            HomeBatchActionBar(...)
        effectiveBatchMode && selectedTab == TabItem.INSPIRE ->
            InspirationBatchActionBar(
                isBatchMode = true,
                selectedInspirationIds = inspirationSelectedIds,
                totalInspirationCount = totalInspirationCount,
                onSelectAll = { inspirationViewModel.selectAllInspirations() },
                onClearSelection = { inspirationViewModel.clearSelection() },
                onDelete = { showInspirationBatchDeleteDialog = true },
                onPin = { inspirationViewModel.batchPinInspirations() }
            )
        else -> CorgiBottomNavigationBar(...)
    }
}
```

### BackHandler

灵感页批量模式时拦截返回键：

```kotlin
BackHandler(enabled = inspirationIsBatchMode) {
    inspirationViewModel.exitBatchMode()
}
```

### 侧滑菜单禁用

批量模式时禁用侧滑菜单（与待办页一致）。

### 状态收集

```kotlin
val isBatchMode by viewModel.isBatchMode.collectAsState()
```

### LazyColumn 渲染

```kotlin
TimelineInspirationItem(
    inspiration = inspiration,
    tags = tags,
    imagePaths = imagePaths,
    formattedTime = formattedTime,
    showDate = item.showDate,
    isPinnedItem = item.isPinned,
    hideDetails = hideDetails,
    isBatchMode = isBatchMode,
    isSelected = selectedInspirationIds.contains(inspiration.id),
    onClick = {
        if (isBatchMode) {
            viewModel.toggleSelection(inspiration.id)
        } else {
            navController.navigate("inspiration_edit/${inspiration.id}")
        }

    onLongClick = if (isBatchMode) {
        {}  // 批量模式禁用长按
    } else {
        {
            longPressedInspiration = inspiration
            showLongPressSheet = true
        }
    }
)
```

### FAB 隐藏

```kotlin
if (!isBatchMode) {
    FloatingActionButton(...) { ... }
}
```

### 批量删除确认弹窗

```kotlin
if (showInspirationBatchDeleteDialog) {
    AlertDialog(
        onDismissRequest = { showInspirationBatchDeleteDialog = false },
        title = { Text("删除选中项") },
        text = { Text("确定要删除已选择的 ${selectedInspirationIds.size} 条灵感吗？\n此操作不可撤销。") },
        confirmButton = {

                viewModel.batchDeleteInspirations()
                showInspirationBatchDeleteDialog = false

        dismissButton = {
            TextButton(onClick = { showInspirationBatchDeleteDialog = false }) { Text("取消") }
        }
    )
}
```
