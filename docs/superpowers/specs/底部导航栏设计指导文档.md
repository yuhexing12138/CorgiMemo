## 第 11 章 灵感记录与特殊日期功能
### 11.2 底部导航栏设计
> 将原右下角FAB按钮替换为底部导航栏，中央编辑按钮点击展开三个气泡。
#### 11.2.1 布局结构
```
┌─────────────────────────────────────────┐
│                                         │
│           [当前页面内容区域]              │
│                                         │
├─────────────────────────────────────────┤
│  📝      💡      ⊕      📅      👤     │  ← 底部导航栏
│  待办    灵感    编辑    日期    我的     │    高度56dp
│  (高亮)                               │    中央⊕突出8dp
└─────────────────────────────────────────┘
```
#### 11.2.2 中央编辑按钮展开效果
```
                ┌─────────┐
                │ 📝待办  │  ← 气泡1：创建待办
                └────┬────┘
     ┌─────────┐    │    ┌─────────┐
     │ 💡灵感  │─── ⊕ ───│ 📅日期  │  ← 气泡2&3
     └─────────┘         └─────────┘
```
#### 11.2.3 底部导航栏详细设计
| 属性 | 值 |
|------|-----|
| **高度** | 56dp |
| **背景** | 白色，顶部1dp #E0E0E0分割线 |
| **Tab项** | 5个：📝待办、💡灵感、⊕编辑、📅日期、👤我的 |
| **中央⊕按钮** | 56dp圆形，暖橙色#FF9A5C，突出导航栏8dp |
| **Tab图标** | 24dp，未选中#999999，选中#FF9A5C |
| **Tab文字** | 12sp，未选中#999999，选中#FF9A5C |
| **切换动画** | 淡入淡出+轻微上移，200ms |
#### 11.2.4 中央编辑按钮交互
| 交互 | 行为 | 动画 |
|------|------|------|
| **点击⊕** | 展开三个气泡 | ⊕旋转45°变为✕，三个气泡从中心弹出（200ms/个，间隔50ms） |
| **点击气泡** | 进入对应编辑页 | 气泡缩放（1.0→1.2→0.8）→ 跳转 |
| **点击✕** | 收起气泡 | 三个气泡依次收回（150ms/个，间隔50ms）→ ✕旋转回⊕ |
| **点击遮罩** | 收起气泡 | 同上 |
| **切换页面** | 自动收起气泡 | 气泡快速收回（100ms） |

#### 11.2.6 检查点（底部导航栏部分）

- [ ] 点击气泡进入对应编辑页

#### 11.2.7 真机验证步骤（底部导航栏部分）
1. 底部应显示5个Tab：📝待办、💡灵感、⊕编辑、📅日期、👤我的
2. 点击⊕ → 应展开三个气泡
3. 点击气泡 → 应进入对应编辑页
4. 切换Tab → 页面应正确切换
