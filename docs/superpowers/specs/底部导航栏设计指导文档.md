### 11.2 底部导航栏设计

> 将原右下角FAB按钮替换为底部导航栏，中央编辑按钮点击展开三个气泡。

#### 11.2.1 布局结构

```
┌─────────────────────────────────────────┐
│                                         │
│           [当前页面内容区域]              │
│                                         │
├─────────────────────────────────────────┤
│  📝      💡      ⊕      📅      👤     │  ← 底部导航栏
│  待办    灵感    编辑    日期    我的     │    高度56dp
│  (高亮)                               │    中央⊕突出8dp
└─────────────────────────────────────────┘
```

#### 11.2.2 中央编辑按钮展开效果

```
                ┌─────────┐
                │ 📝待办  │  ← 气泡1：创建待办
                └────┬────┘
     ┌─────────┐    │    ┌─────────┐
     │ 💡灵感  │─── ⊕ ───│ 📅日期  │  ← 气泡2&3
     └─────────┘         └─────────┘
```

#### 11.2.3 底部导航栏详细设计

| 属性 | 值 | 说明 |
|------|-----|------|
| **高度** | 56dp | 标准底部导航栏高度 |
| **背景** | #FFFFFF（亮色）/ #2A2A2A（深色） | 卡片背景色，顶部1dp分割线 |
| **Tab项** | 5个 | 📝待办、💡灵感、⊕编辑、📅日期、👤我的 |
| **中央⊕按钮** | 56dp圆形 | 主色#FF9A5C，突出导航栏8dp |
| **Tab图标** | 24dp | 未选中#999999，选中主色 |
| **Tab文字** | 12sp Medium | 未选中#999999，选中主色 |
| **切换动画** | 200ms ease-in-out | 淡入淡出+轻微上移 |

**设计规范引用**：
- 颜色：遵循12.1.2颜色规范（主题色+中性色）
- 圆角：中央按钮圆形（50%），导航栏无圆角
- 阴影：中央按钮elevation 6dp，导航栏无阴影

#### 11.2.4 中央编辑按钮交互

| 交互 | 行为 | 动画 | 时长 |
|------|------|------|------|
| **点击⊕** | 展开三个气泡 | ⊕旋转45°变为✕，气泡从中心弹出 | 200ms/个，间隔50ms |
| **点击气泡** | 进入对应编辑页 | 气泡缩放（1.0→1.2→0.8）→ 跳转 | 150ms |
| **点击✕** | 收起气泡 | 气泡依次收回→✕旋转回⊕ | 150ms/个，间隔50ms |
| **点击遮罩** | 收起气泡 | 同上 | 150ms/个 |
| **切换页面** | 自动收起气泡 | 气泡快速收回 | 100ms |

**动效规范引用**（见12.1.7）：
- 微交互：100-150ms
- 元素过渡：200-300ms
- 缓动函数：ease-in-out（切换动画）
