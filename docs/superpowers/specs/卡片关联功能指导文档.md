### 11.6 卡片关联功能（三功能通用）
```
输入 @ 后触发：
┌─────────────────────────────────────────┐
│  @待                                 │   │  ← 输入@后显示搜索框
│   ┌─────────────────────────────┐     │
│   │ 🔍 搜索关联卡片...           │     │  ← 关联选择器
│   ├─────────────────────────────┤     │
│   │ 📝 完成周报          工作   │     │  ← 搜索结果
│   │ 📝 复习考试          学习   │     │    显示类型图标+标题+分类
│   │ 📅 生日              日期   │     │
│   │ 💡 设计灵感          灵感   │     │
│   └─────────────────────────────┘     │
└─────────────────────────────────────────┘
```
#### 11.6.2 关联方式二：搜索选择
在编辑页底部的"关联"区域点击"+"按钮，弹出搜索选择弹窗：
```
┌─────────────────────────────────────────┐
│  关联卡片                        关闭    │
├─────────────────────────────────────────┤
│  🔍 搜索卡片标题...                     │  ← 搜索框
├─────────────────────────────────────────┤
│  📝 待办 (12)                           │  ← 按类型分组
│  ├─ 📝 完成周报                         │
│  ├─ 📝 复习考试                         │
│  └─ 📝 买菜                             │
│  💡 灵感 (5)                            │
│  ├─ 💡 APP新功能构思                     │
│  └─ 💡 读书笔记                         │
│  📅 日期 (3)                            │
│  ├─ 📅 生日                             │
│  └─ 📅 考试                             │
└─────────────────────────────────────────┘
```
#### 11.6.3 关联显示规则
| 位置 | 显示方式 |
|------|---------|
| **编辑页** | 底部"🔗 已关联"区域，显示关联卡片列表，每项可×删除 |
| **列表页** | 卡片底部显示"🔗 关联: @类型:标题"（最多显示1个，多余显示"+N"） |
| **详情页** | 底部"关联卡片"区域，点击可跳转到对应卡片详情 |
#### 11.6.4 关联数据规则
| 规则 | 说明 |
|------|------|
| **关联方向** | 单向关联（A关联B不代表B关联A） |
| **关联数量** | 每张卡片最多关联10张其他卡片 |
| **删除处理** | 删除被关联的卡片时，关联自动解除，不删除源卡片 |
| **类型标识** | 📝待办、💡灵感、📅日期 三种类型图标区分 |

#### 11.6.6 检查点（卡片关联部分）

- [ ] 编辑页输入@可触发关联选择器
- [ ] 编辑页底部显示已关联卡片列表
- [ ] 列表页卡片显示关联提示
- [ ] 删除被关联卡片时自动解除关联

#### 11.6.7 真机验证步骤（卡片关联部分）

1. 在灵感编辑页内容框输入@ → 应弹出关联选择器
2. 搜索并选择一个待办 → 应显示在已关联列表中
3. 保存后返回列表 → 灵感卡片底部应显示"🔗 关联: @待办:标题"
4. 删除被关联的待办 → 灵感的关联应自动解除
