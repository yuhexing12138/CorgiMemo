## 1. 目标

对比参考图与当前实现，修复 4 类 UI 偏差：

1. **状态栏沉浸式适配** — 解决 TopBar 与系统状态栏（时间/信号/电量）重叠问题
2. **TopBar 高度自适应** — TopBar 高度 = 状态栏高度 + 64dp，跨设备一致
3. **卡片尺寸与字体** — 卡片更紧凑（外边距 18→10、圆角 20→12）、字号更接近参考图
4. **品牌文字更新** — Logo "简记事" → "刻记+"，字数徽章内嵌卡片内

## 3. 状态栏沉浸式方案

### 3.1 MainActivity 启用

```kotlin
// MainActivity.kt
override fun onCreate(savedInstanceState: Bundle?) {
    super.onCreate(savedInstanceState)
    enableEdgeToEdge()  // 新增：Android 14+ 推荐 API
    setContent { ... }
}
```

**为什么用 `enableEdgeToEdge()` 而不是手动设置**：
- 自动处理状态栏/导航栏透明
- 自动根据深浅主题调整状态栏文字颜色
- 跨 API 21~35 兼容（由 AndroidX 提供向后兼容）

### 3.2 Scaffold 适配

```kotlin
Scaffold(
    contentWindowInsets = WindowInsets(0, 0, 0, 0),  // 由各 Composable 自行处理 WindowInsets
    containerColor = MaterialTheme.colorScheme.background,
    snackbarHost = { SnackbarHost(snackbarHostState) },
    topBar = { TopBar(...) }
) { paddingValues ->
    // 重要：paddingValues 已经包含状态栏高度，不要再叠加
    Box(modifier = Modifier.fillMaxSize().padding(paddingValues)) { ... }
}
```

## 4. TopBar 高度自适应方案

### 4.1 动态计算状态栏高度

```kotlin
@Composable
private fun TopBar(
    onBack: () -> Unit,
    onCopy: () -> Unit,
    onEdit: () -> Unit,
    onShare: () -> Unit
) {
    val statusBarPadding = WindowInsets.statusBars.asPaddingValues().calculateTopPadding()

    Column(
        modifier = Modifier
            .fillMaxWidth()
            .background(MaterialTheme.colorScheme.background)
    ) {
        // 状态栏占位区域（透明，保留可点击性）
        Spacer(modifier = Modifier.height(statusBarPadding))
        // 实际内容区域（64dp）
        Box(
            modifier = Modifier
                .fillMaxWidth()
                .height(64.dp)
        ) {
            IconButton(...) { /* 返回 */ }
            Box(modifier = Modifier.align(Alignment.CenterEnd).padding(end = 6.dp)) {
                Row {
                    IconButton(...) { /* 复制 */ }
                    IconButton(...) { /* 编辑 */ }
                    IconButton(...) { /* 分享 */ }
                }
            }
        }
    }
}
```

**设计决策**：
- 使用 `Column` 而非单个 `Box`，因为状态栏区域需要独立占位
- 状态栏区域透明保留触摸可点击性（避免系统手势冲突）
- 内容区域仍为 64dp，与原设计一致

### 5.2 字数徽章内嵌实现

**当前实现（Box 外部定位）**：
```kotlin
Box(modifier = modifier.then(graphicsLayerModifier)) {
    Card(...) { Column { /* 标题、正文、Logo */ } }
    // 字数徽章
    Box(modifier = Modifier.align(Alignment.TopEnd).padding(top = 16.dp, end = 28.dp)) { ... }
}
```

**调整后（Column 内部 align End）**：
```kotlin
Card(...) {
    Column(modifier = Modifier.padding(18.dp)) {
        // 字数徽章（横向居右，与内容区分开）
        Box(
            modifier = Modifier
                .align(Alignment.End)
                .background(
                    Color(0xFFE0E0E0).copy(alpha = 0.6f),
                    RoundedCornerShape(12.dp)
                )
                .padding(horizontal = 8.dp, vertical = 4.dp)
        ) {
            Text("${charCount}字", fontSize = 11.sp, color = Color(0xFF666666))
        }
        Spacer(modifier = Modifier.height(8.dp))
        // 标题
        Text(inspiration.title, fontSize = 18.sp, fontWeight = FontWeight.Medium)
        Spacer(modifier = Modifier.height(4.dp))
        // 日期
        Text(formattedDate, fontSize = 12.sp, color = Color(0xFF999999))
        Spacer(modifier = Modifier.height(9.dp))
        // 正文
        Text(inspiration.content, fontSize = 15.sp, lineHeight = 22.sp, color = Color(0xFF666666))
        // 图片列表（如果有）
        if (imagePaths.isNotEmpty()) { ... }
        // 标签
        if (tagsList.isNotEmpty()) { ... }
        // Logo
        Spacer(modifier = Modifier.height(32.dp))
        Row(modifier = Modifier.fillMaxWidth(), horizontalArrangement = Arrangement.Center) {
            Icon(Icons.Outlined.Edit, ...)
            Spacer(modifier = Modifier.width(6.dp))
            Text("- 刻记+ -", fontSize = 13.sp, color = Color(0xFF999999))
        }
    }
}
```

**关键变化**：
- 字数徽章从 Box 外部 `align(TopEnd)` 改为 Column 内部 `align(End)`
- 徽章与下方标题之间增加 8dp Spacer 保持间距
- 整体 Column 顺序：徽章 → 标题 → 日期 → 正文 → 图片 → 标签 → Logo

## 7. 风险评估

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| `enableEdgeToEdge()` 启用后其他页面布局错位 | 中 | 仅在 `InspirationViewScreen` 内 Scaffold 设置 `contentWindowInsets = WindowInsets(0)`，避免全局影响 |
| 状态栏高度计算在低端设备不准确 | 低 | `WindowInsets.statusBars` 由系统提供，已处理 API 差异 |
| 字号调整在长文本时溢出 | 中 | 已用 `TextOverflow.Ellipsis` 或自然换行，长文本会自然换行 |
| 卡片圆角减小影响点击区域 | 低 | 圆角减小不影响点击 hit area（点击区域仍为整个卡片） |
