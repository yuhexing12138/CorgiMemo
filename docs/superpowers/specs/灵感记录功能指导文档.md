#### 11.3.7 真机验证步骤（灵感记录部分）

1. 点击💡灵感Tab → 应进入灵感列表页
2. 灵感列表应按时间线分组
3. 点击中央⊕ → 点击"💡灵感"气泡 → 应进入灵感编辑页
4. 输入标题和内容 → 点击保存 → 应返回列表并显示新灵感
5. 点击灵感卡片 → 应进入详情页
6. 长按灵感卡片 → 应弹出操作菜单（编辑/删除）
7. 删除所有灵感 → 应显示空状态
