# 日期页导航栏日期按钮与纪念日弹窗设计

> 日期：2026-07-14
> 模块：日期页（SpecialDateScreen）
> 状态：已通过 brainstorming，待实施

## 1. 背景与目标

### 1.1 背景

当前 App 的待办页（`HomeScreen`）和灵感页（`InspirationScreen`）顶部导航栏中间都有"日期按钮"（`DatePickerRow` 组件，大号日 25sp + 月份 16sp + ▼），点击后从 topBar 底部全屏展开"日历弹窗"（`TodoCalendarDialog` / `InspirationCalendarDialog`），分为日历区 + 当日列表区。

**日期页**（`SpecialDateScreen`）的导航栏目前没有日期按钮，仅有"📅 日期"标题文字。

### 1.2 目标

在日期页导航栏中间也加入相同的"日期按钮"，点击后弹出对称于待办/灵感页的弹窗：

- **日历区**：日历网格 + 圆点标记有纪念日的日期
- **卡片区**：显示完全匹配选中日期（年月日全等）的纪念日、生日、节日等 SpecialDate 卡片

### 1.3 范围

| 包含 | 不包含 |
|---|---|
| 日期按钮 + 弹窗组件 | 系统节日/法定节日数据源 |
| ViewModel 数据查询方法 | 农历节日 |
| 日历圆点按 DateCategory 色区分 | 弹窗内卡片左滑/置顶/归档操作 |
| 点击卡片跳转详情页 | 弹窗内"今天"快捷按钮 |

## 2. 需求决策（来自 brainstorming 澄清）

| 维度 | 决策 |
|---|---|
| 数据来源 | 仅用户录入的 SpecialDate |
| 匹配规则 | 完全匹配年月日（SpecialDate.targetDate 的 year/month/day 全等） |
| 卡片形态 | 卡片式，复用 `SpecialDateCard` 组件 |
| 点击行为 | 关闭弹窗并跳转到 `SpecialDateDetailScreen` |
| 日历圆点 | 单圆点，按 `DateCategory.color` 取色 |
| 弹窗内卡片操作 | 仅点击跳转，不支持左滑/置顶等操作 |
| 弹窗布局 | 完全对称复用 `TodoCalendarDialog` / `InspirationCalendarDialog` 结构 |

## 3. 选型方案

### 3.1 方案对比

| 方案 | 核心做法 | 优点 | 缺点 |
|---|---|---|---|
| **A. 新建独立组件**（采用） | 在 `date/components/` 下新增 `SpecialDateCalendarDialog.kt`，结构对称于 `TodoCalendarDialog`，卡片项复用 `SpecialDateCard` | 与项目"每屏一 components 目录"约定一致；零回归风险；复用 `SpecialDateCard` + `CalendarMonthView` + `DateWheelPicker` | 与现有两个 dialog 存在结构重复代码 |
| B. 抽象 BaseCalendarDialog\<T\> 模板 | 创建泛型 `BaseCalendarDialog<T>` 接收 countByDate / getItems / itemRenderer | 消除重复代码 | 需重构现有两个 dialog，引入回归风险；与 YAGNI 原则相悖 |
| C. 给 TodoCalendarDialog 加 cardRenderer 参数 | 复用 `TodoCalendarDialog`，增加 cardRenderer 参数 | 不新增文件 | 参数语义混乱，"TodoCalendarDialog"被日期页复用难以理解 |

### 3.2 选择

采用方案 **A**，理由：

1. **对称性高**：与现有 `TodoCalendarDialog` / `InspirationCalendarDialog` 三足鼎立，命名/位置/结构一致
2. **零回归风险**：不改动现有两个组件
3. **复用最大化**：日历区复用 `CalendarMonthView` + `DateWheelPicker`，卡片复用 `SpecialDateCard`
4. **符合 YAGNI**：方案 B 的"未来第 4 个弹窗"是假设需求

### 5.2 内部结构（对称于 `TodoCalendarDialog`）

```
Column(fillMaxSize, padding(top = topPadding), background = surface)
├─ Row: [年月 ▼] ... [×]                      // 顶部行，clickable 切日历/滚轮
├─ Spacer 4dp
├─ when(dateViewMode):
│   ├─ "calendar" → CalendarMonthView(dotColorMap = monthColorMap)
│   └─ "wheel"    → DateWheelPicker
├─ Spacer 4dp
├─ Text "${month}月${day}日 周${weekday} 共 ${dayDates.size} 条"
├─ Spacer 8dp
├─ Box(height=1dp, background=#EEEEEE)        // 分割线
├─ Spacer 8dp
└─ LazyColumn
    ├─ 空态 → "今天还没有纪念日~"
    └─ items(dayDates) → SpecialDateCard(date, onClick = { onDateClick(date) })
```

### 5.3 状态与 LaunchedEffect

```kotlin
var selectedDate by remember { mutableStateOf(initialDate) }
var currentMonth by remember { mutableStateOf(YearMonth.from(initialDate)) }
var dayDates by remember { mutableStateOf<List<DisplayDate>>(emptyList()) }
var monthColorMap by remember { mutableStateOf<Map<Int, Color>>(emptyMap()) }
var dateViewMode by remember { mutableStateOf("calendar") }

LaunchedEffect(currentMonth) {
    monthColorMap = dateCountByDate(currentMonth.year, currentMonth.monthValue)
}

LaunchedEffect(selectedDate) {
    dayDates = getDatesByDate(
        selectedDate.year,
        selectedDate.monthValue,
        selectedDate.dayOfMonth
    )
}
```

### 5.4 `CalendarMonthView` 扩展（仅圆点色）

| 改动 | 代码 |

- 弹窗内不传 `isSimpleMode` / `isBatchMode`（默认 false）
- 不传 `isClickBlocked`（左滑不展开，符合"仅点击跳转"决策）
- `onClick = { onDateClick(date) }` → MainScreen 转发到 `navController.navigate(Screen.SpecialDateDetailWithId.createRoute(date.id))`

## 6. 数据流

### 6.1 ViewModel 新增方法

```kotlin
class SpecialDateViewModel @Inject constructor(
    private val repository: SpecialDateRepository
) : ViewModel() {

    val specialDates: StateFlow<List<SpecialDate>> = repository.allDates

    /**
     * 获取指定月份各日期的圆点颜色映射
     *
     * 同一日期有多张卡时，按 targetDate 升序排序后取首张 category.color
     * 保证相同数据下圆点色稳定。
     *
     * @param year 年
     * @param month 月（1-12）
     * @return Map<day, Color>
     */
    fun getCalendarDateColor(year: Int, month: Int): Map<Int, Color> {
        return specialDates.value
            .filter { item ->
                val cal = Calendar.getInstance().apply { timeInMillis = item.targetDate }
                cal.get(Calendar.YEAR) == year &&
                (cal.get(Calendar.MONTH) + 1) == month
            }
            .groupBy { item ->
                Calendar.getInstance().apply { timeInMillis = item.targetDate }
                    .get(Calendar.DAY_OF_MONTH)
            }
            .mapValues { (_, list) ->
                list.sortedBy { it.targetDate }.first().category.color
            }
    }

    /**
     * 获取完全匹配年月日的纪念日列表
     *
     * @param year 年
     * @param month 月（1-12）
     * @param day 日
     * @return List<DisplayDate> — 按 targetDate 升序排序
     */
    fun getDatesByDate(year: Int, month: Int, day: Int): List<DisplayDate> {
        return specialDates.value
            .filter { item ->
                val cal = Calendar.getInstance().apply { timeInMillis = item.targetDate }
                cal.get(Calendar.YEAR) == year &&
                (cal.get(Calendar.MONTH) + 1) == month &&
                cal.get(Calendar.DAY_OF_MONTH) == day
            }
            .sortedBy { it.targetDate }
            .map { it.toDisplayDate() }
    }
}
```

> 注：`toDisplayDate()` 转换方法已存在于 `SpecialDateViewModel` 中（由现有 `groupedDates` / `pinnedDate` 共用），无需新增。

### 6.2 MainScreen 接入

```kotlin
// 1. 顶层注入 ViewModel（专供弹窗用，与 homeViewModel / inspirationViewModel 模式一致）
val specialDateViewModel: SpecialDateViewModel = hiltViewModel()

// 2. 新增状态
var showSpecialDateCalendar by remember { mutableStateOf(false) }

// 2.1 Tab 切换时重置非当前 tab 的弹窗状态
//     保证切回原 tab 时弹窗默认关闭（不保留上次打开状态）
LaunchedEffect(selectedTab) {
    if (selectedTab != TabItem.DATE) showSpecialDateCalendar = false
    if (selectedTab != TabItem.TODO) showTodoCalendar = false
    if (selectedTab != TabItem.INSPIRE) showInspirationCalendar = false
}

// 3. centerContent 增加 DATE 分支
centerContent = if ((selectedTab == TabItem.INSPIRE
                     || selectedTab == TabItem.TODO
                     || selectedTab == TabItem.DATE) && !effectiveBatchMode) {
    {
        when (selectedTab) {
            TabItem.TODO     -> DatePickerRow(isExpanded = showTodoCalendar, onClick = { showTodoCalendar = !showTodoCalendar })
            TabItem.INSPIRE  -> DatePickerRow(isExpanded = showInspirationCalendar, onClick = { showInspirationCalendar = !showInspirationCalendar })
            TabItem.DATE     -> DatePickerRow(isExpanded = showSpecialDateCalendar,
                                              onClick = { showSpecialDateCalendar = !showSpecialDateCalendar })
            else -> {}
        }
    }
} else null

// 4. Box 末尾追加弹窗渲染（与现有 showTodoCalendar / showInspirationCalendar 块平行）
if (showSpecialDateCalendar && selectedTab == TabItem.DATE) {
    SpecialDateCalendarDialog(
        dateCountByDate = { year, month -> specialDateViewModel.getCalendarDateColor(year, month) },
        getDatesByDate  = { year, month, day -> specialDateViewModel.getDatesByDate(year, month, day) },
        onDateClick     = { date ->
            showSpecialDateCalendar = false  // 关闭弹窗
            navController.navigate(Screen.SpecialDateDetailWithId.createRoute(date.id))
        },
        onDismiss       = { showSpecialDateCalendar = false },
        topPadding      = with(density) { topBarHeightPx.toDp() }
    )
}
```

### 6.3 ViewModel 实例策略

- MainScreen 顶层 `hiltViewModel()` 获取 `specialDateViewModel`，专供弹窗数据查询
- 与 `SpecialDateScreen` 内部的 ViewModel 实例**不共享**（两个实例各自从 `repository.allDates` 加载）
- 数据一致性：两者都基于同一 Room 数据库，弹窗打开时 ViewModel 已加载最新数据，行为一致
- 与现有 `homeViewModel` / `inspirationViewModel` 在 MainScreen 顶层的注入模式一致

## 7. 错误处理与边界情况

| 场景 | 处理 |
|---|---|
| **选中日期无纪念日** | 卡片区显示"今天还没有纪念日~"（与待办/灵感弹窗一致） |
| **用户从未录入任何日期** | 同上，每个日期都为空 |
| **`specialDates.value` 首次未加载** | `getDatesByDate` 返回空列表，显示空态 |
| **滑动切换上/下月** | `CalendarMonthView` 已有 `onMonthChange`；`LaunchedEffect(currentMonth)` 重拉 `dateCountByDate` |
| **选中日期不在当前月** | `CalendarMonthView` 内部已处理（`isPrevOrNextMonth`，圆点不显示） |
| **滚轮模式跨年选择历史纪念日** | `DateWheelPicker` 内部已处理；选中后 `currentMonth` 同步 |
| **跨年查询历史纪念日** | "完全匹配年月日" → 用户可在滚轮模式选历史年份（如查 2020-07-14 的纪念日） |
| **点击 × 关闭** | `onDismiss → showSpecialDateCalendar = false` |
| **点击卡片跳转** | `onDateClick` 先关闭弹窗，再 `navigate` 到详情页 |
| **Tab 切换到其他页** | `LaunchedEffect(selectedTab)` 重置 `showSpecialDateCalendar = false`；条件渲染 `if (showSpecialDateCalendar && selectedTab == TabItem.DATE)` 双重保险，切回原 tab 时弹窗默认关闭 |
| **系统返回键** | 不拦截，与待办/灵感弹窗一致 |
| **同一日期多张卡片** | 圆点色取首张 `category.color`（按 `sortedBy { it.targetDate }` 稳定排序） |
| **数据库异常** | N/A，纯本地 Room，`repository.allDates` 是 StateFlow 自动重发 |
