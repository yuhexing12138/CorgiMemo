## 一、项目背景与目标

### 1.1 背景

- `SpecialDateCardStyleScreen` 的"颜色" Tab 当前是占位 Snackbar(`"颜色功能开发中"`),无实际颜色选择 UI
- 两个日期卡片组件 `OrangeTearOffCard` / `CalendarTearOffCard` 中所有主色调均**硬编码**为 `UiColors.Primary` (`#FF9A5C`) 或 `Color(0xFFFFF8F0)` 等,无法定制
- 参考图(用户提供的 2 张截图)展示:用户从 14 个色圆圈(斜线无颜色 + 12 单色 + 1 彩虹占位)中选色后,大卡片预览实时重新着色

### 1.2 目标

1. **数据模型**:新增 `DateCardColor`(sealed class + serialName),与 `DateCardStyle` 模式对称
2. **DB 字段**:`SpecialDate` 新增 `cardColor: String = "DEFAULT"`,迁移 35→36
3. **卡片参数化**:`OrangeTearOffCard` / `CalendarTearOffCard` / `MiniCalendar` 增 `cardColor` 参数,内部用 helper 函数按 `DateCardColor` 计算 4 类颜色
4. **颜色选择器 UI**:新增 `DateCardColorPicker` 组件(2×7 网格,36dp 圆,选中白边)
5. **屏幕集成**:`SpecialDateCardStyleScreen` 切到 COLOR Tab 时显示 Picker,选色实时更新预览与缩略图
6. **保存**:`SpecialDateCardStyleViewModel.saveNewDate` 增 `cardColor` 参数,落库 `cardColor.serialName`
7. **不读主页**:主页 `SpecialDateCard` 不读 `cardColor`(与 `cardStyle` 一致,后续任务统一实现主页渲染)

### 1.3 不在本设计内

- 主页 `SpecialDateCard` 渲染时读 `cardStyle` / `cardColor`(后续任务)
- 真正的彩虹色(RAINBOW)实现 — 占位 Snackbar
- 4 个被隐藏的样式(蓝色气球/拍立得等)
- 备注/标签/图片/关联等详细字段
- 自定义颜色(用户输入 hex 调色)
- 颜色 tab 的路由参数化(深链支持)— 后续按需扩展

## 二、设计方案选型

### 2.1 数据模型方案对比

| 方案 | 描述 | 优点 | 缺点 | 推荐度 |
|------|------|------|------|--------|
| **方案 A:Sealed Class + serialName(已选)** | `sealed class DateCardColor` + 14 个 `object` 子类,DB 存 `serialName` | 类型安全;`when` 编译期穷尽;与 `DateCardStyle` 模式完全对称;扩展新颜色只需新增 object + 改 when 分支 | 序列化需 `fromSerialName` 解析(成本极低) | ⭐⭐⭐⭐⭐ |
| 方案 B:Int ARGB(可空) | DB 字段 `Int?`,null=无颜色 | 简单直接 | 14 色调色板需硬编码在 picker;新增颜色无类型提示 | ⭐⭐⭐ |
| 方案 C:Int ARGB(永不为空) | DB 字段 `Int`,0=无颜色 | 无空安全顾虑 | 迁移需回填所有行;无类型提示 | ⭐⭐ |

### 2.2 卡片组件颜色接入方案对比

| 方案 | 描述 | 优点 | 缺点 | 推荐度 |
|------|------|------|------|--------|
| **方案 ①:参数化 Composable(已选)** | 卡片 + `cardColor` 参数,内部 `when` 映射 | 精确控制每个元素;纯函数易测;与 sealed class 思路对称 | 需改 5 个组件 | ⭐⭐⭐⭐⭐ |
| 方案 ②:Theme 覆盖 | 用 `CompositionLocalProvider` 覆盖 `MaterialTheme.colorScheme.primary` | 卡片代码改动小 | 需先把 8+ 处 `UiColors.Primary` 改为读主题色;主题色一改全改 | ⭐⭐⭐ |
| 方案 ③:Modifier 重绘 | 外层 `Modifier.drawWithContent` 叠加色块蒙版 | 卡片代码零侵入 | 难精准命中"目标日红圈"等带描边的复合元素 | ⭐⭐ |

### 2.3 选型结论

- **数据模型**:采用方案 A(Sealed Class + serialName),与 `DateCardStyle` 完全对称
- **组件接入**:采用方案 ①(参数化 Composable),保留每张卡每种状态的颜色源显式可控

## 五、`DateCardColor` sealed class 设计

### 5.1 完整定义

```kotlin
package com.corgimemo.app.data.model

import androidx.compose.ui.graphics.Color

/**
 * 日期卡片颜色(主页/QuickCreate 后的颜色选择)
 *
 * 采用 sealed class + serialName 模式,与 DateCardStyle 完全对称:
 * - 类型安全:`when` 编译期穷尽匹配
 * - 持久化:数据库存 serialName 字符串
 * - 解析容错:fromSerialName 失败时 fallback 到 DEFAULT
 *
 * 三种状态:
 * - [Default]: 占位"无颜色"选项(斜线圆),选中后卡片不修改任何颜色
 * - 12 个 [presets] 单色:Blue / SkyBlue / Teal / Green / Lime / Orange / Red / Pink / Purple / Navy / Brown / Black
 * - [Rainbow]: 占位"彩虹"选项(渐变圆),当前选中弹 Snackbar "彩虹色功能开发中"
 *
 * 新增颜色步骤(预留扩展):
 * 1. 在本 sealed class 中添加 `object NewColor : DateCardColor("NEW_COLOR")`
 * 2. 在 [presets] 列表中追加
 * 3. 在 4 个 helper 函数(topBarColor / backgroundColor / bigNumberColor / targetRingColor)中补全 case
 * 4. 在 DateCardColorPicker 的颜色列表中追加
 */
sealed class DateCardColor(val serialName: String) {
    /** 占位"无颜色"选项:选中后卡片不修改任何颜色,使用样式原色 */
    object Default : DateCardColor("DEFAULT")

    // ========== 12 个单色(presets) ==========
    object Blue    : DateCardColor("BLUE")        // 蓝
    object SkyBlue : DateCardColor("SKY_BLUE")    // 天蓝
    object Teal    : DateCardColor("TEAL")        // 青绿
    object Green   : DateCardColor("GREEN")       // 绿
    object Lime    : DateCardColor("LIME")        // 黄绿
    object Orange  : DateCardColor("ORANGE")      // 橙
    object Red     : DateCardColor("RED")         // 红
    object Pink    : DateCardColor("PINK")        // 粉
    object Purple  : DateCardColor("PURPLE")      // 紫
    object Navy    : DateCardColor("NAVY")        // 深蓝
    object Brown   : DateCardColor("BROWN")       // 棕
    object Black   : DateCardColor("BLACK")       // 黑

    /** 占位"彩虹"选项:当前选中弹 Snackbar,功能后续实现 */
    object Rainbow : DateCardColor("RAINBOW")

    companion object {
        val DEFAULT: DateCardColor = Default

        /** 真实可选的 12 种单色(不含 Default 与 Rainbow) */
        val presets: List<DateCardColor> = listOf(
            Blue, SkyBlue, Teal, Green, Lime, Orange,
            Red, Pink, Purple, Navy, Brown, Black
        )

        /**
         * 从 serialName 解析为 DateCardColor 实例
         * @param name serialName 字符串(null/非法值 fallback 到 DEFAULT)
         */
        fun fromSerialName(name: String?): DateCardColor = when (name) {
            Default.serialName  -> Default
            Blue.serialName     -> Blue
            SkyBlue.serialName  -> SkyBlue
            Teal.serialName     -> Teal
            Green.serialName    -> Green
            Lime.serialName     -> Lime
            Orange.serialName   -> Orange
            Red.serialName      -> Red
            Pink.serialName     -> Pink
            Purple.serialName   -> Purple
            Navy.serialName     -> Navy
            Brown.serialName    -> Brown
            Black.serialName    -> Black
            Rainbow.serialName  -> Rainbow
            else                -> DEFAULT
        }
    }
}
```

### 5.3 4 个 helper 函数(挂在 DateCardColor 文件内,顶层函数)

```kotlin
// ==================== 顶部条颜色(橙撕专用) ====================
/**
 * 橙撕卡片顶部 60dp 撕页条背景色
 * - DEFAULT → UiColors.Primary(#FF9A5C),与现有硬编码一致
 * - 非 DEFAULT → 调色板对应色
 * - RAINBOW → 留给绘制时不调用此函数,顶层用 Brush.sweepGradient 覆盖
 */
fun topBarColor(color: DateCardColor): Color = when (color) {
    DateCardColor.Default -> Color(0xFFFF9A5C)            // UiColors.Primary
    DateCardColor.Blue   -> Color(0xFF3F5BFF)
    DateCardColor.SkyBlue -> Color(0xFF1E9CFF)
    DateCardColor.Teal   -> Color(0xFF26C7B7)
    DateCardColor.Green  -> Color(0xFF4CAF50)
    DateCardColor.Lime   -> Color(0xFF8BC34A)
    DateCardColor.Orange -> Color(0xFFFF9A5C)
    DateCardColor.Red    -> Color(0xFFFF5252)
    DateCardColor.Pink   -> Color(0xFFEC407A)
    DateCardColor.Purple -> Color(0xFF7E57C2)
    DateCardColor.Navy   -> Color(0xFF1A237E)
    DateCardColor.Brown  -> Color(0xFF6D4C41)
    DateCardColor.Black  -> Color(0xFF212121)
    DateCardColor.Rainbow -> Color(0xFFFF9A5C)            // 彩虹实际不调用,fallback
}

// ==================== 卡片背景色(两种样式共用) ====================
/**
 * 卡片背景色
 * - DEFAULT:
 *   - 橙撕 → 白色(等同 MaterialTheme.colorScheme.surface)
 *   - 日历 → 米色 #FFF8F0
 * - 非 DEFAULT → 调色板对应色(根据样式选深浅版,见 [backgroundColor] 第二个重载)
 * - RAINBOW → 留给绘制时用 Brush 覆盖
 */
fun backgroundColor(color: DateCardColor, style: DateCardStyle): Color = when (color) {
    DateCardColor.Default -> when (style) {
        DateCardStyle.OrangeTearOff -> Color.White
        DateCardStyle.CalendarTearOff -> Color(0xFFFFF8F0)
    }
    else -> backgroundColorFilled(color)                   // 调色板原色作为背景
}

/**
 * 调色板色作为卡片背景(用于非 DEFAULT 状态)
 * - 浅色(Blue/SkyBlue/Teal/Green/Lime/Orange/Pink)→ 用调色板原色
 * - 深色(Red/Purple/Navy/Brown/Black)→ 用调色板原色(用户已选深色,接受)
 */
private fun backgroundColorFilled(color: DateCardColor): Color = topBarColor(color)

// ==================== 大数字颜色(两种样式共用) ====================
/**
 * 大数字 64sp Bold 文字色
 * - DEFAULT → UiColors.Primary(#FF9A5C)
 * - 深色背景(Navy / Black / Brown / Purple / Red)→ 两种样式都用 White,以保证与同色背景的对比度
 * - 其他浅色 → 调色板对应色(与 topBarColor 一致)
 */
fun bigNumberColor(color: DateCardColor, style: DateCardStyle): Color = when (color) {
    DateCardColor.Default -> Color(0xFFFF9A5C)
    DateCardColor.Navy,
    DateCardColor.Black,
    DateCardColor.Brown,
    DateCardColor.Purple,
    DateCardColor.Red -> Color.White
    else -> topBarColor(color)
}

// ==================== 目标日红圈颜色(日历专用) ====================
/**
 * MiniCalendar 目标日 1.5dp 描边圈颜色
 * - DEFAULT → 现有红色 #FFFF8A80
 * - 非 DEFAULT → 调色板对应色
 * - RAINBOW → 留给绘制时用 Brush 覆盖
 */
fun targetRingColor(color: DateCardColor): Color = when (color) {
    DateCardColor.Default -> Color(0xFFFF8A80)
    else -> topBarColor(color)
}
```

### 5.4 对比度规则(实现阶段验证清单)

为保证"大数字"在卡片背景(与选中色同色)上始终可读,以下 5 个深色 **无论样式** 都需要将大数字用白色:

| 调色板色 | 橙撕大数字 | 日历大数字 | 说明 |
|----------|-----------|-----------|------|
| Navy | **White** | **White** | 背景为深蓝,白字 |
| Black | **White** | **White** | 背景为近黑,白字 |
| Brown | **White** | **White** | 背景为深棕,白字 |
| Purple | **White** | **White** | 背景为深紫,白字 |
| Red | **White** | **White** | 背景为深红,白字 |

其余 7 个浅色 + Orange 调色板:橙撕与日历大数字均用调色板色(蓝/天蓝/青绿/绿/黄绿/橙/粉)。

### 6.1 `OrangeTearOffCard` 修改

@Composable
fun OrangeTearOffCard(
    title: String,

    modifier: Modifier = Modifier,
    isThumbnail: Boolean = false,
    cardColor: DateCardColor = DateCardColor.DEFAULT,  // ← 新增,默认 DEFAULT 保持现有行为
    onShareClick: (() -> Unit)? = null,
    onCalendarClick: (() -> Unit)? = null,
    onCardClick: (() -> Unit)? = null
) {
    val displayTitle = title.ifBlank { "未命名" }
    val topSectionHeight = if (isThumbnail) 18.dp else 60.dp
    val holeSize = if (isThumbnail) 4.dp else 16.dp

/**

 *
 * 布局(参考图):
 * - 第 1 行:Default(斜线无颜色) + Blue + SkyBlue + Teal + Green + Lime + Orange
 * - 第 2 行:Red + Pink + Purple + Navy + Brown + Black + Rainbow(渐变占位)

@Composable
fun DateCardColorPicker(
    selected: DateCardColor,
    onSelect: (DateCardColor) -> Unit,
    onRainbowClick: () -> Unit,
    modifier: Modifier = Modifier
) {
    // 第 1 行 7 个 + 第 2 行 7 个
    val row1: List<DateCardColor> = listOf(
        DateCardColor.Default,
        DateCardColor.Blue, DateCardColor.SkyBlue, DateCardColor.Teal,
        DateCardColor.Green, DateCardColor.Lime, DateCardColor.Orange
    )
    val row2: List<DateCardColor> = listOf(
        DateCardColor.Red, DateCardColor.Pink, DateCardColor.Purple,
        DateCardColor.Navy, DateCardColor.Brown, DateCardColor.Black,
        DateCardColor.Rainbow
    )

    Column(
        modifier = modifier
            .fillMaxWidth()
            .height(IntrinsicSize.Min),  // 高度由内容撑开
        verticalArrangement = Arrangement.spacedBy(8.dp)
    ) {
        // 第 1 行
        ColorRow(items = row1, selected = selected, onSelect = onSelect, onRainbowClick = onRainbowClick)
        // 第 2 行
        ColorRow(items = row2, selected = selected, onSelect = onSelect, onRainbowClick = onRainbowClick)
    }

@Composable
private fun ColorRow(
    items: List<DateCardColor>,
    selected: DateCardColor,
    onSelect: (DateCardColor) -> Unit,
    onRainbowClick: () -> Unit
) {
    Row(
        modifier = Modifier
            .fillMaxWidth()
            .padding(horizontal = 16.dp),
        horizontalArrangement = Arrangement.spacedBy(8.dp)
    ) {
        items.forEach { color ->
            ColorCircle(
                color = color,
                isSelected = color == selected,
                onClick = {
                    if (color == DateCardColor.Rainbow) onRainbowClick() else onSelect(color)
                },
                modifier = Modifier.weight(1f)
            )
        }
    }
}

@Composable
private fun ColorCircle(
    color: DateCardColor,
    isSelected: Boolean,
    onClick: () -> Unit,
    modifier: Modifier = Modifier
) {
    Box(
        modifier = modifier
            .aspectRatio(1f)                          // 直径 = 容器宽度(weight 等分 7 列后约 36dp)
            .then(
                if (isSelected) Modifier
                    .border(2.dp, Color.White, CircleShape)
                else Modifier
            )
            .padding(if (isSelected) 2.dp else 0.dp)  // 选中态给描边留出 2dp 间隙,避免覆盖
            .clip(CircleShape)
            .clickable { onClick() },
        contentAlignment = Alignment.Center
    ) {
        // 内部画"颜色盘"本身(36dp - 2dp×2 = 32dp)
        Box(
            modifier = Modifier
                .fillMaxSize()
                .clip(CircleShape)
                .background(
                    when (color) {
                        DateCardColor.Default -> Color.White  // 斜线圆底
                        DateCardColor.Rainbow -> rainbowBrush()
                        else -> topBarColor(color)            // 单色
                    }
                )
        ) {
            // Default 圆:在白底上画一条对角红线
            if (color == DateCardColor.Default) {
                Canvas(modifier = Modifier.fillMaxSize()) {
                    val strokeWidth = 2.dp.toPx()
                    drawLine(
                        color = Color(0xFFFF5252),
                        start = Offset(0f, 0f),
                        end = Offset(size.width, size.height),
                        strokeWidth = strokeWidth
                    )
                }
            }
        }
    }
}

/**
 * 模拟彩虹的 sweepGradient brush
 * 使用 6 段色:红/橙/黄/绿/蓝/紫
 */
private fun rainbowBrush(): Brush = Brush.sweepGradient(
    colors = listOf(
        Color(0xFFFF5252),  // 红
        Color(0xFFFF9A5C),  // 橙
        Color(0xFFFFEB3B),  // 黄
        Color(0xFF4CAF50),  // 绿
        Color(0xFF3F5BFF),  // 蓝
        Color(0xFF7E57C2),  // 紫
        Color(0xFFFF5252)   // 闭环
    )
)

## 十四、附录:配色与资源

| 名称 | 值 | 用途 |
|------|-----|------|
| Default(无颜色) | 白底 + 红斜线 | 颜色选择器第 1 行第 1 个 |
| Rainbow(彩虹占位) | sweepGradient 6 色 | 颜色选择器第 2 行第 7 个 |
| 选中态描边 | `#FFFFFFFF` 2dp | 所有圆圈选中态 |
| 选中态内边距 | 2dp | 给描边留出间隙,避免覆盖 |
| 圆圈直径 | 36dp(由 weight+aspectRatio 算出) | 单个色圆 |
| 行间距 | 8dp | 2 行之间 |
| 圆与圆间距 | 8dp | spacedBy |
| 行内边距 | horizontal=16dp | 与 tab 切换条对齐 |
| 调色板色值 | 见 5.2 表 12 个 ARGB | 卡片颜色源 |
