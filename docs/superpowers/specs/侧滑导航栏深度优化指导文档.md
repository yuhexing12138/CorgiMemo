#### 📱 真机验证步骤

**验证侧滑导航栏**
1. 从左边缘右滑 → 侧滑导航栏应打开
2. 应显示头像、昵称、"分组管理"标题、"最近删除"入口
3. 应显示"全部(N)"、"未分类(N)"等分组列表
4. 点击"添加分组" → 应弹出输入弹窗 → 输入名称 → 新分组出现在列表中
5. 长按自定义分组 → 应弹出"更多"弹窗（置顶/编辑/删除）
6. 点击遮罩层 → 侧滑栏应关闭
7. 侧滑栏点击自定义分组 → 应进入分组列表页
8. 标题应显示分组名称 + 暖橙色装饰下划线
9. 仅显示该分组的待办
10. 点击 ← 返回 → 应回到首页
