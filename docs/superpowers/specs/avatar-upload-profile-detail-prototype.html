设计要点

  • 大头像120dp 圆形 + 主色背景 + 白色首字母（与改造前 72dp 同 UserAvatar 组件，缩放一致）
  • 表项56dp 高 + 12dp 圆角 + 1dp 阴影 + 主色文字标签 + 灰色占位（已对齐 UI设计规范 §12.1.2）
  • BottomSheet4 个来源横向等宽，主色圆形图标 + 中文标题 + 灰色描述
  • 预设库3 列网格复用柯基动作帧，selected 状态用主色 3dp 边框
  • 裁剪半透明黑色蒙版 + 白色圆形边框（box-shadow 9999px 实现穿透蒙版）

交互流

  • 屏幕 ①: 抽屉/Profile 头卡头像 → 跳本页
  • 屏幕 ①: 点击大头像 → 弹屏幕 ②
  • 屏幕 ②: 选"拍照"或"相册" → 弹屏幕 ④ 裁剪
  • 屏幕 ②: 选"预设库" → 弹屏幕 ③
  • 屏幕 ④: 拖动/缩放 → 点"使用" → 头像更新 + 关闭所有弹层

响应式

  • 屏幕宽度 ≥ 360dp：底部导航 4 项完整显示
  • 屏幕宽度 < 360dp：BottomSheet 改为全屏 Dialog
  • 预设库网格：宽度 < 360dp 时降为 2 列（本期固定 3 列，后续优化）
