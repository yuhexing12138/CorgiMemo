# 日期卡片尺寸重设设计文档

## 概述

严格按照参考图对两种日期详情卡片（橙色撕页样式、日历撕页样式）的尺寸大小和内部元素比例进行重新设置，同时添加参考图中的新装饰元素，并调整状态栏图标颜色行为。

## 背景与目标

当前两种日期卡片使用固定 dp/sp 值，内部元素布局为 `SpaceEvenly` 均匀分布，与参考图的视觉比例不一致。本次重构后：
- 卡片使用固定宽高比，与参考图一致
- 内部元素使用 `BoxWithConstraints` 按卡片总高百分比分配，比例精确
- 添加参考图中的新装饰元素（虚线、点点装饰、顶部波浪等）
- 状态栏图标保持黑色，不随背景色改变

## 设计方案

### 整体架构

使用 **`BoxWithConstraints`** 包裹卡片内容，通过 `maxHeight` 获取卡片总高度，所有内部元素尺寸/间距均按**卡片总高度的百分比**计算。

**卡片整体宽高比：**
- 橙色撕页样式：**宽:高 = 4:5**
- 日历撕页样式：**宽:高 = 1:2**

实现方式：在卡片最外层 `Modifier` 上添加 `.aspectRatio(ratio)`，父容器宽度决定卡片宽度，高度按比例自动计算。

## 橙色撕页样式（OrangeTearOffCard）

### 高度分配（占卡片总高百分比）

| 区域 | 占比 | 说明 |
|------|------|------|
| 顶部撕页条 | 20% | 彩色条 + 2 个白色圆孔 |
| 白色内容区 | 80% | 距离文字 + 大数字 + 标题 + 底部装饰 |
| 底部波浪 | 内含于内容区底部（约 3%） | 撕页波浪效果 |

### 元素尺寸详情

| 元素 | 尺寸/位置 | 说明 |
|------|----------|------|
| 顶部撕页条高度 | 卡片总高 × 20% | 含顶部圆角区域 |
| 圆孔直径 | 顶部条高度 × 27% | 两个白色圆孔 |
| 圆孔左右边距 | 卡片宽度 × 10% | 距左右边缘 |
| 距离文字字号 | 卡片总高 × 4% | 灰色 #999999 |
| 大数字字号 | 卡片总高 × 30% | Bold，跟随 cardColor |
| 标题字号 | 卡片总高 × 5% | Medium，跟随 cardColor |
| 底部波浪高度 | 卡片总高 × 3% | 撕页波浪效果 |
| 底部点点装饰 | 距左下角 8%，5~6 个小圆点 | 虚线样式圆点 |
| 右下角图标 | 卡片总高 × 5% | 分享 + 日历，灰色 #999999 |

### 内部布局（白色内容区）

使用 `Column` + `Spacer(weight)` 精确控制比例：
- 顶部间距（距离文字）
- 大数字（居中，占主要空间）
- 底部间距（标题 + 底装饰）

### 新增元素：底部点点装饰

- 位置：卡片底部左侧
- 样式：5~6 个小圆点组成虚线排列
- 圆点大小：约 3dp
- 颜色：浅灰 #CCCCCC

## 日历撕页样式（CalendarTearOffCard）

### 高度分配（占卡片总高百分比）

| 区域 | 占比 | 说明 |
|------|------|------|
| 顶部波浪 | 3% | 顶部撕页波浪边（新增） |
| 顶部标题区 | 7% | "2026年7月" + 三点指示器 |
| 月历区 | 50% | 星期表头 + 5 行日期 + 虚线 |
| 标题区 | 7% | 用户输入标题 |
| 大数字区 | 20% | 天数大数字 |
| 距离文字区 | 7% | "距离 yyyy/MM/dd 还有" |
| 底部波浪 | 3% | 底部撕页波浪边 |
| 缓冲/间距 | 3% | 各区域间微调 |

### 元素尺寸详情

| 元素 | 尺寸/位置 | 说明 |
|------|----------|------|
| 顶部波浪高度 | 卡片总高 × 3% | 顶部撕页波浪（新增） |
| 月份标题字号 | 卡片总高 × 4.5% | Medium，深灰 |
| 三点指示器圆点 | 卡片总高 × 1.2% | 浅灰 #CCCCCC |
| 星期表头字号 | 卡片总高 × 3.5% | 浅灰 #999999 |
| 日期数字字号 | 卡片总高 × 3.8% | 正常日期深灰，目标日跟随 cardColor |
| 目标日圆圈描边 | 1.5dp | 跟随 cardColor |
| 虚线分割线 | 0.5dp 宽，4dp 实 + 4dp 虚 | 浅灰 #CCCCCC |
| 标题字号 | 卡片总高 × 4.5% | Medium，固定 onSurface |
| 大数字字号 | 卡片总高 × 18% | Bold，跟随 cardColor |
| 距离文字字号 | 卡片总高 × 3.5% | 灰色 #999999 |
| 底部波浪高度 | 卡片总高 × 3% | 底部撕页波浪 |

### 新增元素：虚线

共 3 处虚线：

1. **星期表头下方虚线**：表头与第一行日期之间
2. **每行日期之间虚线**：5 行日期，共 4 条横向虚线
3. **月历下方主分割虚线**：月历区域与下方标题/数字之间的分割线

虚线样式：
- 高度：0.5dp
- Pattern：4dp 实线 + 4dp 空白，循环
- 颜色：浅灰 #CCCCCC

### 新增元素：顶部波浪

- 位置：卡片最顶部
- 形状：波浪形撕页边
- 高度：卡片总高 × 3%
- 实现：自定义 Shape，与底部波浪 `WavyBottomShape` 类似，改为顶部波浪
- 注意：月份标题区内容向下偏移一个波浪高度，避免被波浪裁剪

## MiniCalendar 改动

### 新增参数

- `dividerColor: Color` — 虚线颜色（默认 #CCCCCC）
- `showDividers: Boolean` — 是否显示虚线（默认 true）

### 结构改动

- 星期表头下方添加 0.5dp 虚线
- 每行日期之间添加 0.5dp 虚线
- 整个日历整体下方不添加虚线（由外部卡片层级控制月历下方主分割虚线）

## 状态栏颜色调整

### 当前行为
根据背景色亮度自动切换状态栏图标深浅（`isAppearanceLightStatusBars` 动态变化）。

### 修改后行为
状态栏图标**始终保持深色（黑色）**，无论背景色是什么颜色。

### 实现方式
- `isAppearanceLightStatusBars` 固定为 `true`
- 移除动态亮度判断逻辑

---

## 缩略图版（StyleSelector 缩略图）

策略：保持当前 `isThumbnail` 模式，所有尺寸等比缩放。缩略图版不需要严格按比例，保持当前 80dp × 120dp 的固定尺寸，元素按比例简化缩放。

| 文件 | 改动程度 | 说明 |

| `OrangeTearOffCard.kt` | 较大 | 加 `aspectRatio(4/5)` + `BoxWithConstraints` 比例尺寸 + 底部点点装饰 |

| `SpecialDateCardStyleScreen.kt` | 小 | 移除状态栏图标自动变色逻辑，固定深色 |

## 风险与注意事项

1. **文字大小兼容性**：所有文字使用 `sp` 单位，按比例计算时注意转换
2. **波浪裁剪内容偏移**：顶部波浪会裁剪内容，内容需向下偏移波浪高度，避免被裁掉
3. **虚线绘制性能**：虚线使用 `Canvas` + `PathEffect.dashPathEffect` 实现
4. **缩略图版一致性**：缩略图版保持固定尺寸，完整版使用比例系统
