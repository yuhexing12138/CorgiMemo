### 悬浮柯基按钮交互

| 交互                         | 行为                               | 动画                                            |
| ---------------------------- | ---------------------------------- | ----------------------------------------------- |
| **点击**               | 进入柯基详情页                     | 缩放动画（1.0→1.2→1.0，300ms）→ 跳转到详情页 |
| **长按（>500ms）**     | 进入拖动模式，可拖动到任意位置     | 按钮放大1.2倍+阴影加深，跟随手指移动            |

| **拖动结束**           | 停留在释放位置                     | 按钮回弹动画（轻微缩放1.0→0.9→1.0，200ms）    |
| **快速滑动（<200ms）** | 识别为滑动操作，触发快捷功能       | 根据滑动方向触发对应功能                        |
| **滑动左**             | 快速添加待办（BottomSheet形式）    | 按钮向左滑出，显示快捷添加表单                  |

| **空闲时**             | 触摸互动小动画（眨眼/歪头/摇尾巴） | 每30秒随机播放一个（概率30%）                   |
| **完成待办时**         | 跳跃庆祝动画                       | 小跳跃（高度20dp，300ms）                       |
| **情绪变化时**         | 表情变化                           | 淡入淡出切换表情（200ms）                       |
| **靠近屏幕边缘**       | 自动贴边吸附                       | 向最近边缘滑动吸附（50%屏幕宽度为界）           |

#### 拖动行为详细说明

**拖动模式触发条件：**

- 长按按钮超过500ms → 进入拖动模式
- 拖动模式下按钮视觉变化：放大1.2倍、阴影加深、透明度0.9

**拖动边界限制：**

```
┌─────────────────────────────────────────┐
│  ↑ 安全区域上边界 (状态栏高度+8dp)       │
│                                         │
│    ┌─────┐                              │
│    │ 🐕  │  ← 可拖动区域                 │
│    └─────┘                              │
│                                         │
│  ↓ 安全区域下边界 (导航栏高度+88dp)      │
└─────────────────────────────────────────┘
```

**位置记忆机制：**

- 使用 DataStore 保存按钮位置（相对于屏幕的百分比坐标）
- 下次启动时恢复到保存的位置
- 如果保存位置超出当前屏幕范围，自动调整到最近的有效位置

**贴边吸附逻辑：**

- 拖动结束后，判断按钮中心点距离左右边缘的距离
- 如果距离左边缘 < 屏幕宽度的50% → 吸附到左边缘（距边缘16dp）
- 如果距离右边缘 ≤ 屏幕宽度的50% → 吸附到右边缘（距边缘16dp）
- 吸附动画：300ms 弹性动画
