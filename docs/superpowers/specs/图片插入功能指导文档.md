### 11.5 图片插入功能（三功能通用）

> 待办、灵感、特殊日期三个功能均支持插入图片，无数量限制。

#### 11.5.1 图片选择器设计

```
┌─────────────────────────────────────────┐
│  选择图片                        完成    │  ← 顶部操作栏
├─────────────────────────────────────────┤
│  ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐  │
│  │      │ │      │ │      │ │      │  │  ← 相册网格
│  │      │ │      │ │      │ │      │  │    3列网格
│  │  ✓   │ │      │ │  ✓   │ │      │  │    已选显示✓
│  └──────┘ └──────┘ └──────┘ └──────┘  │
│  ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐  │
│  │      │ │      │ │      │ │      │  │
│  │      │ │      │ │      │ │      │  │
│  │      │ │      │ │      │ │      │  │
│  └──────┘ └──────┘ └──────┘ └──────┘  │
├─────────────────────────────────────────┤
│  📷 拍照    已选择 2 张                  │  ← 底部操作栏
└─────────────────────────────────────────┘
```

#### 11.5.2 图片功能规则

| 规则 | 说明 |
|------|------|
| **数量限制** | 无限制，可插入任意数量图片 |
| **图片来源** | 相册选择 + 相机拍照 |
| **图片压缩** | 自动压缩至最大2048px，JPEG质量85% |
| **存储位置** | APP内部存储 `pictures/` 目录 |
| **缩略图** | 编辑页显示96×96dp缩略图，详情页显示全尺寸 |
| **预览** | 点击图片可全屏预览，支持左右滑动切换 |
| **删除** | 编辑页中每张图片右上角有×删除按钮 |
| **排序** | 长按图片可拖拽排序 |

#### 11.5.4 检查点（图片插入部分）
- [ ] 三个功能编辑页均可插入图片
- [ ] 图片选择器支持相册选择+相机拍照
- [ ] 图片自动压缩
- [ ] 点击图片可全屏预览
- [ ] 图片可删除
#### 11.5.5 真机验证步骤（图片插入部分）
1. 在待办编辑页点击添加图片 → 应弹出图片选择器
2. 选择图片 → 图片应显示在编辑页中
3. 点击图片 → 应全屏预览
4. 点击图片× → 图片应删除
5. 在灵感编辑页和日期编辑页重复以上步骤
