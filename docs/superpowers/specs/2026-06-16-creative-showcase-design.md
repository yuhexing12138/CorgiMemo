# 刻记+ 创意展示网页设计文档

> 面向 Trae AI 创意大赛评委的创意展示网页

## 设计目标

将「刻记+」PRD中的创意亮点以沉浸式网页形式展示给评委，着重展示四大创意点：柯基陪伴系统、身份自适应设计、三位一体记录系统、节日节气关怀。

## 设计方案：沉浸式故事 + 页签导航

### 整体架构

- **5个页签**：概览 / 柯基陪伴 / 身份自适应 / 三位一体 / 节日关怀
- **深色沉浸背景**：深蓝渐变 + 柯基暖色光晕
- **毛玻璃卡片**：backdrop-filter 模糊效果
- **故事叙述**：用户场景故事穿插展示

### 各页签内容

#### Tab 1: ✨ 概览
- 产品Slogan + 柯基光晕动画
- 三大核心价值：陪伴感 / 高效性 / 温暖感
- 目标用户群体卡片（上班族/学生/老年）
- 产品愿景引言

#### Tab 2: 🐕 柯基陪伴系统（重点展示）
- 柯基形象体系：8种姿态展示
- 表情系统：8种表情切换演示
- 情绪系统：情绪值计算公式 + 状态变化动画
- 触摸互动演示：单击/双击/长按的不同反应
- 装扮系统：节日装扮 + 成就装扮展示
- 成长系统：4个等级形态变化
- 用户故事：小林的晨间问候场景

#### Tab 3: 👤 身份自适应设计
- 三种身份对比：上班族/学生/老年模式并排展示
- 界面风格差异：色调/布局/图标/分类对比
- 问候语差异：不同身份的晨间问候示例
- 专属功能：番茄钟/学习专注/用药提醒
- 实时切换演示：点击切换身份看界面变化

#### Tab 4: 📋 三位一体记录系统
- 三大功能关系图：待办 ↔ 灵感 ↔ 日期的关联可视化
- 待办管理：创建/完成/子任务拆分演示
- 灵感记录：时间线展示 + 标签系统 + 富文本
- 特殊日期：天数卡片 + 倒计时/正计时
- 跨功能关联：灵感→待办、日期→待办的转化演示

#### Tab 5: 🎉 节日节气关怀
- 节日列表：传统节日/二十四节气/现代节日
- 柯基换装展示：春节/端午/中秋/冬至装扮
- 专属问候语示例：冬至场景完整演示
- 用户故事：小林的冬至关怀场景
- 公益价值：从孤独到温暖的情感连接

### 视觉增强

- 深蓝渐变背景（#1a1a2e → #16213e → #0f3460）
- 柯基暖色光晕（#FF9A5C drop-shadow）
- 粒子特效：星星/爱心/雪花飘动动画
- 毛玻璃卡片：backdrop-filter: blur
- 页签切换动画：300ms ease-in-out

### 配色方案

基于APP设计规范，深色模式增强：
- 主色：#FF9A5C（暖阳橙）
- 主色浅：#FFB380
- 背景渐变：#1a1a2e → #0f3460
- 卡片背景：rgba(255,255,255,0.06)
- 文字主色：#E8E6E3
- 文字次色：rgba(255,255,255,0.5)
- 强调色：#7EC8A0（成功）、#FFB74D（警告）

### 技术方案

- 单文件HTML，所有CSS/JS内联
- 纯CSS动画（柯基光晕、粒子飘动、卡片入场、页签切换）
- 响应式设计：桌面(1200px+)/平板(768px+)/手机(375px+)
- 无外部依赖，离线可运行
- HTML5语义化标签
