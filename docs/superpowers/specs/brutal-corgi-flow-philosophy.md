# Brutal Corgi Flow

## 野兽派柯基流场 · 算法哲学

Brutal Corgi Flow 是一场精心策划的视觉叛乱——在杂志风的优雅秩序中，释放野兽派的力量与柯基的治愈。这不是随机的装饰，而是经过无数次迭代精调的大师级实现：每一个参数都承载着对"力量感与治愈感如何共存"这一命题的深度回答。Perlin 噪声流场赋予粒子有机的、如溪流般的运动轨迹，而旋转量化到 45° 倍数、2-3px 粗黑边框、4-6px 硬偏移阴影则像一把手术刀，在流动中切出棱角。这种有机与硬边的张力，正是 Brutal Corgi Flow 的灵魂——它拒绝成为温顺的背景动画，也拒绝成为粗暴的视觉噪音，而是在两极之间找到了那个经过无数次迭代才能抵达的精确平衡点。

流场是有机的，但野兽派是几何的——这一矛盾不是缺陷，而是引擎。当柯基粒子沿 Perlin 噪声的等势线滑行时，它们的角度被强制量化到 45° 的网格上，仿佛一只真实的柯基在方格纸上奔跑：身体的轨迹是自由的，但姿态被网格捕获。粗边框将柔软的柯基轮廓从背景中暴力切割出来，硬偏移阴影（零模糊、纯色偏移）在粒子的右下方投下另一个自己——一个更硬、更绝对的影子。色块叠加在粒子背后铺开野兽派色板（橙红、暖黄、薄荷绿、深黑），像海报设计中的油墨套印错位。这些处理不是随意的风格叠加，而是经过无数次迭代精调的视觉语法：每一条边框的粗细、每一个阴影的偏移量、每一次旋转的量化步长，都是大师级手感对"刚刚好"这一标准的反复逼近。

参数系统是 Brutal Corgi Flow 的暗面——用户看到的是 15-25 只柯基在屏幕上悠然飘动，看不到的是背后每一个数值都经历了数十轮 A/B 对比后的落锤定音。noiseScale 0.003 让流场的波纹尺度恰好匹配屏幕宽度，既不显得局促也不显得空旷；flowSpeed 0.8 让粒子以"散步而非奔跑"的节奏穿越视口；透明度锁定在 0.15-0.35 的窄区间，确保粒子始终是"氛围的呼吸"而非"视线的争夺"。粒子数量被严格限制在 15-25 个——因为柯基 PNG 图片比抽象几何更重，每一个都是完整的形象而非一个点，多一只就是多一份视觉主张。这不是保守，而是克制，是经过无数次迭代精调后对"少即是多"的野兽派诠释：每一只柯基都必须被看见，而不是淹没在同伴的汪洋中。

每一个粒子都是一枚微型宣言。它说：我是一只柯基，但我有粗边框；我在流场中飘荡，但我的朝向被 45° 网格锚定；我透明到几乎不存在，但我的阴影硬得像铁。这 11 种姿态——tilt 的歪头、stand 的挺立、sleep 的蜷缩、sit 的端庄、shy 的闪躲、sad 的低落、run 的冲刺、roll 的翻滚、proud 的昂首、lie 的慵懒、wag 的欢快——不是随机的表情包，而是经过无数次迭代精调的情绪光谱，每一帧都在无声地诉说：力量可以温柔，秩序可以呼吸，野兽派可以治愈。当一只透明度 0.2 的 shy 柯基以 45° 量化角从屏幕左下角滑入，它的粗边框和硬阴影在杂志风的页面上划出一道不服从的轨迹——这就是 Brutal Corgi Flow 的全部意义。

Brutal Corgi Flow 从不卡顿。流畅不是技术指标，而是哲学承诺——野兽派的力量感来自确定性，来自每一个粒子都精确地出现在它该出现的位置、以它该有的角度、带着它该有的边框和阴影。卡顿是失控，而失控是野兽派最不能容忍的。Perlin 噪声的连续性保证了运动的丝滑，旋转量化在帧间通过最近邻取整实现无跳变过渡，透明度在粒子生命周期两端以余弦曲线淡入淡出。整个系统像一台精密的瑞士钟表，而表盘上跑动的不是指针，是柯基。这是经过无数次迭代精调的大师级实现——它看起来毫不费力，因为所有的力都已经被精确地分配到了每一个像素、每一帧、每一度旋转之中。
