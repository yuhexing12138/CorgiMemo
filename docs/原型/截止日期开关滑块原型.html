改动说明

核心改动

维度 改前 改后
截止日期行 UI 「截止日期」标签 + 日期芯片 + 时间芯片（始终显示） 「截止日期」标签 + Switch 开关（农历同款，默认关闭）
默认值 截止日期 = 提醒时间（始终有值） 截止日期 = null（不选择）
选择交互 直接点芯片切换日历/时间视图 先打开开关 → 下方展开日期+时间芯片 → 点芯片选择
关闭开关 — 清空已设截止日期，再次打开需重选
确定按钮 始终传 dueDateMillis（自动修正早于提醒时间的值） 开关关闭时传 null；打开时传选定值（保留自动修正）

布局结构（改后）

  • 提醒时间行（不变）
  • 截止日期行：标签 + Switch 开关
  • ↳ 开关打开时：下方展开「日期芯片 + 时间芯片」
  • 重复提醒行（不变）
  • 农历行（不变，作为开关样式参考）
  • 分割线 + 日历/时间滚轮 + 取消/确定按钮

状态说明

① 默认：开关关闭，状态标签「未设置」，确定时截止日期 = null
② 打开开关：下方展开芯片，默认日期/时间 = 提醒时间
③ 选中截止日期：点日期芯片激活日历，选中日期后日历下方有橙点标记

设计依据：开关样式完全复用农历行（同款 Material 3 Switch，主色 #FF9A5C 轨道）。配色、字号、圆角均遵循 UI设计规范 12.1.2 / 12.1.5。展开动画 250ms ease，符合 12.1.7.1 元素过渡 200-300ms。

交互演示

点击上方状态切换按钮查看三种状态；也可直接点击弹窗内的截止日期开关自由切换。
