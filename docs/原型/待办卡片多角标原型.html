🔑 关键设计细节
1. 角标高度统一为 19dp：height: 19px + font-size: 11px + line-height: 11px，与分享图分类角标严格一致。

2. 起点为竖条右侧 4dp：padding-left: 20px（卡片） = 4dp 竖条 + 16dp 内容，所以角标行 margin-left: 0 即可。

3. 圆角仅给最右：通过 .badge-row > .badge:last-child { border-bottom-right-radius: 16px; } 实现，中间角标全 0 圆角以无缝拼接。

4. 视觉降权（已完成态）：背景 rgba(0,0,0,0.03) + 文字 #AAAAAA，与分享图分类角标一致。

5. 角标顺序固定：分组 → 图片 → 语音 → 关联（图片在语音前，与元数据行旧版保持一致）。

6. 移除元数据行附件/关联：v1.1 起，AttachmentCountsRow 和 RelationCountChip 不再在元数据行渲染，统一由角标行接管。
❓ 待确认问题
Q1：当 todo 没有任何分组/附件/关联时，badge-row 是否需要保留？
→ 建议：所有角标都隐藏时，badge-row 也隐藏（display: none），不留空隙。

Q2：附件角标的"图片"和"语音"是分两个独立角标还是合并？
→ 当前设计：分两个角标（🖼️2 和 🎤1），更清晰。
→ 备选：合并为 📎3 一个角标（占用空间更小）。

Q3：分组角标 emoji（📂）是否保留？
→ 当前设计：保留（视觉更直观）。
→ 备选：去掉 emoji，只显示文字（如 v1.0 旧版）。
