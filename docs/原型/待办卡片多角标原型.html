🔑 关键设计细节
1. 角标高度统一为 19dp：height: 19px + font-size: 11px + line-height: 11px，与分享图分类角标严格一致。

2. Material Icons 选择（避免 emoji 跨平台不一致）：
• 分组 → Icons.Default.Folder（文件夹）
• 附件 → Icons.Default.AttachFile（回形针，图片+语音合并计数）
• 关联 → Icons.Outlined.Link（链接）

3. 起点为竖条右侧 4dp：padding-left: 20px（卡片） = 4dp 竖条 + 16dp 内容。

4. 圆角仅给最右：.badge-row > .badge:last-child { border-bottom-right-radius: 16px; }。

5. 视觉降权（已完成态）：背景 rgba(0,0,0,0.03) + 文字 #AAAAAA。

6. 角标顺序固定：分组 → 附件 → 关联（图片+语音合并后顺序更简洁）。

7. 空态处理：所有角标都隐藏时，badge-row 完全不渲染（display: none），不留空白。
