重设计说明

核心改动

维度 旧版 新版
视觉风格 橙色渐变 header + 暖色内容区 还原 App 白色卡片 + 优先级竖条 + 边框
优先级 文字徽章（高/中/低/无） 左侧 4dp 竖条 + 1.5dp 边框（三联视觉）
标题 22sp 居中粗体 16sp 左对齐 Medium + 圆形复选框
子待办 ❌ 无 ✅ 展开列表（复选框+标题+进度）
时间信息 只有 startDate 提醒时间 + 截止日期（芯片样式）
图片附件 ❌ 无 ✅ 缩略图行（最多3张 + "+N"）
品牌区 header 大面积渐变 顶部精简品牌条 + 底部水印
尺寸 360×560 dp（固定高度） 360 宽 × 自适应高度（内容决定）

布局结构（从上到下）

  • 品牌顶部条：🐕 logo + "刻记+" + "柯基备忘录" + 日期胶囊
  • 主待办卡片：还原 App 内 TodoListItem 样式
    • 左侧 4dp 优先级竖条
    • 1.5dp 优先级边框
    • 圆形复选框 + 标题 + 进度(2/4)
    • 元数据行：分类 + 附件数 + 关联数
    • 时间行：提醒📅 + 截止🏁（芯片样式）
    • 子待办列表：复选框 + 标题
    • 图片缩略图行
  • 品牌水印：分割线 + "🐕来自 CorgiMemo · 让待办变得可爱"

交互演示

优先级切换：点击上方圆点切换，竖条和边框颜色同步变化
场景切换：有子待办 / 无子待办 / 已完成态

设计原则：分享图片 = 品牌条 + App 卡片原样复刻 + 水印。让接收者看到和 App 内完全一致的卡片样式，降低认知差异。图片宽度 360dp（1080px @3x），高度随内容自适应。
