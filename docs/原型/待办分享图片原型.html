v5.9 源代码修复：竖条 clipPath 圆心与边框外侧圆心严格对齐

变更点 说明
修复 clipPath 圆心 源代码 addRoundRect 圆角 9dp，圆心 = (clipPath left + 9dp, clipPath top + 9dp)。
原 clipPath top = startY → 圆心 Y = startY+9dp，与边框外侧圆心 Y=startY+8.25dp 差 0.75dp，导致竖条在圆角处与边框外侧错位 0.75dp。
对齐 圆心位置 clipPath 范围上下左右各扩 barOffset = 0.75*density，让 clipPath 圆心 = 边框外侧圆心 = (padding + 8.25dp, startY + 8.25dp)。
竖条在圆角处紧贴外侧边缘，视觉上无错位。
原型 CSS 限制 原型中 .todo-card border-radius: 9px 让 .todo-card 圆角 = 9px 半径 + 9px 圆心距边角。
与源代码"边框外侧 9dp 半径 + 8.25dp 圆心距边角"差 0.75px 圆心位置，视觉上几乎不可见（< 1px 差异）。
CSS 无法直接控制 border-radius 圆心位置，接受此细微差异。

v5.8 竖条左边缘严格对齐边框外侧边缘

变更点 说明
对齐 竖条左边缘 竖条 left: 0 改为 left: -1.5px，外延 1.5px = 边框宽度。
效果：竖条左边缘 = 边框外侧边缘（严格对齐）
调整 竖条宽度 竖条 width 从 4px 改为 calc(4px + 1.5px) = 5.5px。
效果：竖条右边缘保持 4px（距 .todo-card 左边 4px），与分类角标 left: 4px 无缝拼接
自动裁剪 外延部分 竖条外延 [-1.5px, 0] 范围在 .todo-card box 外，被 overflow: hidden 自动裁剪。
视觉上只看到 [0, 4px] 范围内的竖条，但竖条"逻辑位置"已与边框外侧对齐
对齐 源代码 源代码中竖条 drawRect left = padding - 0.75*density，barOffset = strokeWidth/2 = 0.75dp。
clipPath 范围扩大 barOffset，让竖条外延部分在 clipPath 内可见。
原型用 overflow: hidden 实现相同视觉效果（CSS 与 Canvas 机制不同但结果一致）

v5.7 边框内侧截标签背景：边框完全压住角标/时间标签

变更点 说明
核心 边框位置调整 将 CSS 边框从 border: 1.5px solid（在 box 外）改为 box-shadow: inset 0 0 0 1.5px（在 box 内 1.5px）。
视觉上边框 [7.5dp, 9dp] 范围（距卡片外边）与源代码 Canvas stroke 中心线 8.25dp 圆角完全对齐。
修复 角标透出 核心目标：边框完全压住角标/时间标签背景色，不透出。
原理：边框 z-index=4 在角标 z-index=3 之上，box-shadow inset 1.5px 范围与角标 [0, 7.5dp] 范围分离 → 视觉上边框 [7.5, 9dp] 范围内是纯边框色，角标只在 [0, 7.5dp] 范围内可见。
对齐 源代码 源代码使用 clipPath 圆角 7.5dp 裁剪角标，让角标 [7.5, 9dp] 范围不可见。
原型用 box-shadow: inset + z-index 4 实现相同视觉效果。

布局结构（从上到下）

  • 品牌顶部条：App 启动图标（柯基歪头图 32dp）+ "刻记⁺"（上标）+ "治愈系柯基记录应用" + 分享时间胶囊（含年）
  • 主待办卡片：
    • 分类角标（卡片左上角贴边，仅右下圆角 9dp）
    • 优先级角标（卡片右上角贴边，仅左下圆角 9dp）
    • 时间标签（卡片左下角贴边，仅右上圆角 9dp，与分类对称）
    • 左侧 4dp 优先级竖条（充满卡片高度，clipPath 自动裁剪）
    • 1.5dp 优先级边框

    • 子待办列表（唯一内容区）
  • 品牌水印：分割线 + "来自 刻记⁺ · 让待办变得可爱"

Z 顺序（从下到上）

  1. 卡片白色背景（z=0）
  2. 三个角标：分类角标（左上） + 优先级角标（右上） + 时间标签（左下）（z=3）
  3. 优先级边框（::after 伪元素模拟，z=4）覆盖所有角标背景色
  4. 左侧优先级竖条（z=5，最上层）同时覆盖所有角标和优先级边框

设计要点

三个角标（左上分类 + 右上优先级 + 左下时间）形成"三角对称"布局： 分类角标三边贴边 + 仅右下圆角； 优先级角标右/上/下三边贴边 + 仅左下圆角； 时间标签左/下两边贴边 + 仅右上圆角； 三个角标的圆角与卡片圆角均严格一致（9dp）。

边框层叠技巧：CSS box border 不能被 z-index 覆盖在子元素之上， 所以优先级边框用 ::after 伪元素 + z-index:4 模拟， 让边框"压"在角标背景色边缘上。竖条 z-index 最高（5），同时覆盖边框和角标。
