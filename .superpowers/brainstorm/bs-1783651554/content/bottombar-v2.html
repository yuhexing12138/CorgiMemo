交互行为说明

展开/收起规则

  • 点击 ⋮ → 展开上行工具栏
  • 再次点击 ⋮ → 收起工具栏
  • 点击工具栏内格式按钮 → 不收起
  • 点击相机/麦克风/#/@ → 不收起
  • 编辑器输入/光标移动 → 不收起
  • 软键盘弹出/收起 → 不收起

⋮ 激活态视觉

  • 收起态：浅橙背景 #FFE0C0
  • 展开态：实心橙背景 #FF9A5C + 白字
  • 明确指示当前工具栏状态

# 与 @ 按钮行为

  • # 位置按钮：点击触发位置提醒弹窗（showLocationPopup），与编辑器内输入 # 行为一致
  • @ 关联按钮：点击触发关联选择弹窗（showMentionPopup），与编辑器内输入 @ 行为一致
