设计细节

布局方案：中心辐射式

推荐。待办管理作为中心枢纽节点，灵感记录和特殊日期环绕分布。体现「待办管理是核心」的产品定位。使用曲线连接替代直线，更优雅地展示双向关系。

节点角色颜色
📋 待办管理核心枢纽橙色
💡 灵感记录创意输入层绿色
📅 特殊日期时间锚点层黄色

动画方案：GSAP 三阶段

分三个阶段动态展示功能间关系：

  1. 入场阶段（0-1.5s）：中心节点 scale+fadeIn → 卫星节点依次 stagger 弹入
  2. 连线阶段（1.5-2.5s）：SVG path strokeDashoffset 动画绘制曲线连线
  3. 流动阶段（循环）：沿路径流动的小圆点粒子表示数据流向，hover 节点时加速

交互与响应式

增强信息传达效率：

  • Hover 节点：高亮该节点所有连线，显示关系说明 tooltip
  • 点击节点：脉冲扩散效果，展示该功能的详细描述
  • 移动端适配：≤768px 时改为垂直链式布局，保持动画效果
  • prefers-reduced-motion：禁用粒子动画，保留静态连线
