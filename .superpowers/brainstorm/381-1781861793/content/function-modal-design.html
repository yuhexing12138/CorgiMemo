方案对比

✓ 方案A：居中模态弹窗（推荐）

点击节点 → 背景半透明遮罩淡入 + 内容卡片从中心弹性缩放弹出。关闭时反向动画。

  • ✓ 动画：backdrop fadeIn(0.25s) + modal scale+fadeIn(cubic-bezier 弹性)
  • ✓ 关闭方式：X按钮 / 点击遮罩 / ESC键
  • ✓ 三大模块内容完整保留，原样嵌入弹窗
  • ✓ 移动端全屏展示，体验一致
  • ✓ 支持 prefers-reduced-motion 降级为无动画切换

方案B：右侧滑入面板

点击节点 → 从右侧滑入面板，宽度占屏幕 80%。

  • 适合宽屏，移动端也友好
  • 可看到部分背景上下文
  • 但与野兽派整体风格不够统一

实施要点

步骤操作
① 隐藏原章节给第12/13/15章添加 style="display:none" 或 CSS 类隐藏
② 创建弹窗容器在 body 末尾添加 3 个 <div class="func-modal"> 容器
③ 迁移内容将三章节的 innerHTML 复制到对应弹窗容器内
④ 绑定点击事件关系图节点 click → 对应弹窗显示 + 动画触发
⑤ HyperFrames 风格动画GSAP timeline: backdrop→modal header→modal body stagger 入场
⑥ 关闭交互X按钮/遮罩/ESC → 反向退出动画 → 隐藏弹窗
