设计章节 2/6：页面详细设计（修订版）

已根据反馈调整：使用项目现有帧动画、APP名称改为"刻记+"

可用柯基帧动画资源（13种）

SIT 坐立 STAND 站立 LIE 趴卧 RUN 奔跑 WINK 眨眼 WAG 摇尾巴 TILT 歪头 ROLL 打滚 SLEEP 睡觉 SAD 难过 PROUD 骄傲 SHY 害羞 WORRY 担心

通过 AnimationResourceManager.getAnimationFrames(type) 获取帧资源，用 FrameAnimation 组件播放

阶段1：个性化设置

🐶
Step 1
欢迎页（修订）

WelcomePage（改进版）

布局：
- 顶部：柯基 WAG（摇尾巴）帧动画
  尺寸 120dp，循环播放，友好欢迎感
- 中部：APP名称 "刻记⁺"
  使用 BaselineShift.Superscript 实现 + 上标
- 底部：slogan "记录生活，刻下每一刻"
- 底部按钮："开始体验"
技术实现：
FrameAnimation(
  frames = AnimationResourceManager
    .getAnimationFrames(WAG),
  fps = 12,
  modifier = Modifier.size(120.dp)
)

+上标实现：
Text(buildAnnotatedString {
  append("刻记")
  withStyle(SpanStyle(
    baselineShift = Superscript,
    fontSize = 14.sp)) { append("+") }
})
