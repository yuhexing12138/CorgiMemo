### 步骤 4：悬浮柯基按钮组件

**新建文件**: `app/src/main/java/com/corgimemo/app/ui/components/FloatingCorgiButton.kt`

**组件设计**:

```
FloatingCorgiButton(
    onClick: () -> Unit,           // 点击进入柯基详情页
    onPositionChanged: (Float, Float) -> Unit,  // 位置变化回调
    initialPosition: Pair<Float, Float>?,       // 初始位置（从DataStore恢复）
    modifier: Modifier = Modifier
)
```

**功能清单**:
1. **视觉**: 48×48dp，使用 FrameAnimation 播放柯基头像帧动画，elevation 8dp 阴影
2. **呼吸动画**: 缓慢放大缩小循环（2s周期），使用 `infiniteRepeatable` + `tween`
3. **点击交互**: 缩放动画（1.0→1.2→1.0，300ms）→ 触发 onClick
4. **长按拖动**: 长按>500ms 进入拖动模式
   - 拖动模式视觉：放大1.2倍 + 阴影加深 + 透明度0.9
   - 使用 `pointerInput` + `detectDragGestures` 实现拖动
   - 边界限制：安全区域内（状态栏高度+8dp 到 导航栏高度+88dp）
5. **贴边吸附**: 拖动结束后判断中心点位置，吸附到最近边缘（距边缘16dp），300ms弹性动画
6. **位置记忆**: 通过 `onPositionChanged` 回调保存位置到 DataStore
7. **空闲动画**: 每30秒随机播放（30%概率），使用 `LaunchedEffect` + `delay`
   - 眨眼：切换到 WINK 帧动画
   - 歪头：切换到 TILT 帧动画
   - 摇尾巴：切换到 WAG 帧动画
8. **完成庆祝**: 暴露 `triggerCelebration()` 方法，触发小跳跃（高度20dp，300ms）

**实现要点**:
- 使用 `Modifier.pointerInput` 处理长按+拖动手势
- 使用 `Modifier.offset` 控制位置
- 使用 `animateFloatAsState` / `Animatable` 控制各种动画
- 贴边吸附使用 `Animatable.animateTo` + `spring` 弹性动画

### 步骤 5：柯基详情页

**新建文件**: `app/src/main/java/com/corgimemo/app/ui/screens/corgi/CorgiDetailScreen.kt`

**页面结构**:

- 使用 `LinearProgressIndicator` 显示情绪值进度条

- 使用 `TaskDailyStatsRepository` 获取数据
- 点击可跳转到统计详情页（`Screen.Stats`）

#### 5.7 互动按钮区

## 假设与决策

1. **悬浮按钮使用 FrameAnimation**：用户选择帧动画柯基头像，复用现有 `FrameAnimation` + `AnimationType` 系统
2. **柯基详情页复用 HomeViewModel**：详情页的数据（corgiData、currentMood、currentPose等）已在 HomeViewModel 中管理，直接复用避免重复
3. **悬浮按钮位置用百分比存储**：适配不同屏幕尺寸，存储相对屏幕的百分比坐标
4. **空状态简化**：根据规格文档，空状态不再显示柯基动画，仅文字+CTA按钮
5. **互动按钮动画复用**：抚摸/喂食/玩耍的动画效果复用现有 AnimationType（WAG/PROUD/RUN等）
