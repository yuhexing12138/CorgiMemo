## 三、详细实现方案

### 3.1 触觉反馈实现

#### 技术选型

* **Android 12+ (API 31+)**：使用 `HapticFeedback` 或 `VibrationEffect`

* **Android 12 以下**：使用传统 `Vibrator.vibrate()`

#### 在 InteractiveCorgi.kt 中添加

```kotlin
// 获取 Vibrator 服务
val vibrator = LocalContext.current.getSystemService(Context.VIBRATOR_SERVICE) as Vibrator

// 触觉反馈函数
fun performHapticFeedback(type: InteractionType) {
    when (type) {
        InteractionType.SINGLE_CLICK -> {
            // 短震动 50ms
            if (Build.VERSION.SDK_INT >= Build.VERSION_CODES.O) {
                vibrator.vibrate(VibrationEffect.createOneShot(50, VibrationEffect.DEFAULT_AMPLITUDE))
            } else {
                vibrator.vibrate(50)
            }
        }
        InteractionType.DOUBLE_CLICK -> {
            // 中等震动 100ms
            if (Build.VERSION.SDK_INT >= Build.VERSION_CODES.O) {
                vibrator.vibrate(VibrationEffect.createOneShot(100, VibrationEffect.DEFAULT_AMPLITUDE))
            } else {
                vibrator.vibrate(100)
            }
        }
        InteractionType.LONG_CLICK -> {
            // 脉冲震动：等待50ms + 震动50ms + 等待50ms + 震动50ms + 等待50ms + 震动50ms
            val pattern = longArrayOf(50, 50, 50, 50, 50, 50)
            if (Build.VERSION.SDK_INT >= Build.VERSION_CODES.O) {
                vibrator.vibrate(VibrationEffect.createWaveform(pattern, -1))
            } else {
                vibrator.vibrate(pattern, -1)
            }
        }
    }
}
```

#### 需要的权限

在 `AndroidManifest.xml` 中添加：

```xml
<uses-permission android:name="android.permission.VIBRATE" />
```

### 3.2 音效反馈实现

#### 方案一：使用 SoundPool（推荐，适合短音效）

```kotlin
// SoundManager.kt
class SoundManager(private val context: Context) {
    
    private var soundPool: SoundPool? = null
    private var wagSoundId: Int = 0
    
    init {
        if (Build.VERSION.SDK_INT >= Build.VERSION_CODES.LOLLIPOP) {
            soundPool = SoundPool.Builder()
                .setMaxStreams(5)
                .setAudioAttributes(
                    AudioAttributes.Builder()
                        .setUsage(AudioAttributes.USAGE_GAME)
                        .setContentType(AudioAttributes.CONTENT_TYPE_SONIFICATION)
                        .build()
                )
                .build()
        } else {
            soundPool = SoundPool(5, AudioManager.STREAM_MUSIC, 0)
        }
        
        // 加载音效
        wagSoundId = soundPool?.load(context, R.raw.wag_sound, 1) ?: 0
    }
    
    fun playWagSound() {
        soundPool?.play(wagSoundId, 1.0f, 1.0f, 1, 0, 1.0f)
    }
    
    fun release() {
        soundPool?.release()
        soundPool = null
    }
}
```

#### 方案二：直接在 InteractiveCorgi.kt 中使用 MediaPlayer（简单方案）

```kotlin
val context = LocalContext.current
val mediaPlayer = remember { MediaPlayer.create(context, R.raw.wag_sound) }

// 播放音效
if (soundEnabled) {
    mediaPlayer.start()
}
```

#### DataStore 存储音效开关状态

修改 `CorgiPreferences.kt`：

```kotlin
companion object {
    private val CORGI_NAME = stringPreferencesKey("corgi_name")
    private val IS_FIRST_LAUNCH = booleanPreferencesKey("is_first_launch")
    private val SOUND_ENABLED = booleanPreferencesKey("sound_enabled")  // 新增
    private val HAPTIC_ENABLED = booleanPreferencesKey("haptic_enabled")  // 新增
}

// 新增获取和设置方法
val soundEnabled: Flow<Boolean> = dataStore.data
    .map { preferences -> preferences[SOUND_ENABLED] ?: true }

val hapticEnabled: Flow<Boolean> = dataStore.data
    .map { preferences -> preferences[HAPTIC_ENABLED] ?: true }

suspend fun setSoundEnabled(enabled: Boolean) {
    dataStore.edit { preferences -> preferences[SOUND_ENABLED] = enabled }
}

suspend fun setHapticEnabled(enabled: Boolean) {
    dataStore.edit { preferences -> preferences[HAPTIC_ENABLED] = enabled }
}
```

### 3.3 设置页面实现

#### 新增 `SettingsScreen.kt`

```kotlin
@Composable
fun SettingsScreen(
    navController: NavController,
    corgiPreferences: CorgiPreferences,
    modifier: Modifier = Modifier
) {
    val soundEnabled by corgiPreferences.soundEnabled.collectAsState(initial = true)
    val hapticEnabled by corgiPreferences.hapticEnabled.collectAsState(initial = true)
    val coroutineScope = rememberCoroutineScope()
    
    Scaffold(
        topBar = {
            TopAppBar(
                title = { Text("设置") },
                navigationIcon = {
                    IconButton(onClick = { navController.popBackStack() }) {
                        Icon(Icons.AutoMirrored.Filled.ArrowBack, "返回")
                    }
                }
            )
        }
    ) { paddingValues ->
        Column(
            modifier = modifier
                .padding(paddingValues)
                .fillMaxSize()
        ) {
            // 音效开关
            SwitchSettingItem(
                title = "音效反馈",
                description = "触摸柯基时播放音效",
                checked = soundEnabled,
                onCheckedChange = { enabled ->
                    coroutineScope.launch {
                        corgiPreferences.setSoundEnabled(enabled)
                    }
                }
            )
            
            // 触觉开关
            SwitchSettingItem(
                title = "触觉反馈",
                description = "触摸柯基时震动",
                checked = hapticEnabled,
                onCheckedChange = { enabled ->
                    coroutineScope.launch {
                        corgiPreferences.setHapticEnabled(enabled)
                    }
                }
            )
        }
    }
}
```

### 3.4 视觉强化实现

#### 边缘光晕效果组件

```kotlin
@Composable
fun GlowOverlay(
    show: Boolean,
    modifier: Modifier = Modifier
) {
    AnimatedVisibility(
        visible = show,
        enter = fadeIn(),
        exit = fadeOut()
    ) {
        Box(
            modifier = modifier
                .fillMaxSize()
                .background(
                    Brush.verticalGradient(
                        colors = listOf(
                            Color(0xFFFF9A5C).copy(alpha = 0.3f),
                            Color.Transparent,
                            Color.Transparent,
                            Color(0xFFFF9A5C).copy(alpha = 0.3f)
                        )
                    )
                )
        ) {
            // 左右边缘光晕
            Box(
                modifier = Modifier
                    .fillMaxHeight()
                    .width(50.dp)
                    .background(
                        Brush.horizontalGradient(
                            colors = listOf(
                                Color(0xFFFF9A5C).copy(alpha = 0.4f),
                                Color.Transparent
                            )
                        )
                    )
            )
            Box(
                modifier = Modifier
                    .fillMaxHeight()
                    .width(50.dp)
                    .align(Alignment.CenterEnd)
                    .background(
                        Brush.horizontalGradient(
                            colors = listOf(
                                Color.Transparent,
                                Color(0xFFFF9A5C).copy(alpha = 0.4f)
                            )
                        )
                    )
            )
        }
    }
}
```

## 五、潜在依赖和考虑

### 5.1 音效资源

| 项目   | 说明                                   |
| ---- | ------------------------------------ |
| 格式   | MP3 或 OGG（推荐 OGG，Android 原生支持）       |
| 时长   | 建议 0.5-1 秒（短音效）                      |
| 放置目录 | `app/src/main/res/raw/wag_sound.mp3` |

### 5.2 版本兼容性

| API 级别                | 震动 API            | 音频 API                        |
| --------------------- | ----------------- | ----------------------------- |
| Android 8.0 (API 26+) | `VibrationEffect` | `SoundPool.Builder`           |
| Android 5.0 (API 21+) | 传统 `vibrate()`    | `SoundPool.Builder`           |
| Android 5.0 以下        | 传统 `vibrate()`    | 传统 `SoundPool(int, int, int)` |

### 5.3 性能考虑

| 功能   | 性能影响 | 优化建议                       |
| ---- | ---- | -------------------------- |
| 震动   | 无影响  | 使用系统服务，无需优化                |
| 音效   | 低    | 使用 SoundPool，预加载音效         |
| 视觉效果 | 低    | 使用 AnimatedVisibility，性能良好 |

## 六、风险处理

| 风险       | 影响    | 规避方案                   |
| -------- | ----- | ---------------------- |
| 音效资源缺失   | 应用崩溃  | 使用 try-catch 或检查资源是否存在 |
| 震动权限问题   | 震动不生效 | 震动权限是普通权限，无需动态申请       |
| 开关状态不同步  | 设置不生效 | 使用 StateFlow 实时更新      |
| 音效播放失败   | 无声音   | 添加 try-catch，记录日志      |
| 旧版本兼容性问题 | 功能失效  | 使用版本判断，兼容旧版本           |
