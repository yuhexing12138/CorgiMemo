### 2.3 详细功能设计

#### 功能 1：情绪与姿态联动

**设计说明**：
- `PoseManager.getPoseForMood(mood)` 已实现，但 `HomeViewModel` 中未使用
- 需要在情绪变化时，自动更新姿态

**实现步骤**：
1. 在 `HomeViewModel` 中，情绪更新时调用 `PoseManager.getPoseForMood()`
2. 在 `initPoseAndMood()` 中也应用情绪姿态
3. 任务完成时临时切换到庆祝姿态，完成后恢复情绪姿态

#### 功能 2：等级形态显示

**设计说明**：
- 等级阶段：柯基宝宝(1-3)、柯基少年(4-6)、柯基青年(7-9)、柯基大师(10+)
- 不同阶段应有不同的视觉表现

**实现步骤**：
1. 在 `InteractiveCorgi` 中新增 `level` 参数
2. 根据 `LevelManager.getLevelStage(level)` 判断阶段
3. 为不同阶段设置不同的尺寸比例：
   - 柯基宝宝：80% 大小
   - 柯基少年：90% 大小
   - 柯基青年：100% 大小
   - 柯基大师：100% 大小 + 特殊光晕效果

#### 功能 3：装扮显示

**设计说明**：
- 装扮状态存储在 `CorgiData.currentOutfit` 中
- 需要在柯基上方叠加装扮图标

**实现步骤**：
1. 新建 `OutfitOverlay` 组件
2. 根据 `outfitId` 显示对应的装扮图标
3. 使用 Emoji 作为临时占位：
   - 学士帽：🎓
   - 领带：👔
   - 皇冠：👑
   - 天使翅膀：🪽
   - 披风：🧥
4. 定位在柯基头部上方

#### 功能 4：完整的柯基互动区域

**设计说明**：
- 创建一个卡片式组件，整合柯基展示和信息
- 顶部：柯基动画（含装扮）
- 中部：柯基名字 + 等级 + 情绪状态
- 底部：经验进度条 + 问候语

**实现步骤**：
1. 创建 `CorgiInteractionCard` 组件
2. 使用 Card 包裹所有内容
3. 内部使用 Column 布局
4. 渐变背景 + 圆角 16dp

#### 功能 5：任务完成庆祝动画

**设计说明**：
- 当前已有 `CelebrationOverlay`，可以增强效果
- 柯基切换到站立姿态（已实现）
- 显示庆祝消息

**实现步骤**：
1. 增强 `CelebrationOverlay`，添加更多动画元素
2. 在庆祝期间显示 "太棒了！" 等消息
3. 柯基切换到 `CorgiPose.STAND`（已实现）

## 五、风险与注意事项

### 5.1 潜在风险

| 风险 | 说明 | 缓解措施 |
|------|------|----------|
| 性能问题 | 动画叠加可能导致性能下降 | 使用 AnimatedVisibility 控制显隐 |
| 布局适配 | 不同屏幕尺寸的布局问题 | 使用约束布局和百分比 |
| 状态同步 | ViewModel 状态与 UI 不同步 | 使用 collectAsState 确保响应式 |

### 5.2 注意事项

1. **保持向后兼容**：确保现有功能不被破坏
2. **遵循 Material3 规范**：使用 MaterialTheme 颜色和圆角
3. **支持深色模式**：颜色方案要同时支持明/暗模式
4. **添加中文注释**：所有新函数和复杂逻辑都要有注释
