## 1. 项目概述

### 1.1 产品定位与标签

**「刻记+」**是一款定位于**学习工作**与**社会公益**两大领域的治愈系生活管理应用。

| 标签维度           | 定位说明                                           | 核心价值                                         |
| ------------------ | -------------------------------------------------- | ------------------------------------------------ |
| **学习工作** | 以三大记录功能为核心，服务学生、上班族、自由职业者 | 高效管理任务、灵感、重要日期，提升学习与工作效率 |
| **社会公益** | 以柯基陪伴系统为载体，关注心理健康与情感陪伴       | 缓解孤独焦虑、培养良好习惯、提供心灵疗愈         |

**产品使命**：不仅帮助用户管理生活事务，更致力于成为用户日常生活中的温暖陪伴者，特别关注学生、上班族和老年群体的情感需求与心理健康。

### 1.2 产品愿景

> **"不会忘记的，是每一天陪你的小短腿"**

在快节奏的现代生活中，待办事项、灵感火花、重要日期往往散落各处——冷冰冰的列表、易逝的想法、被遗忘的纪念日。

**「刻记+」**想要改变这一切。

我们相信，**生活记录可以是有温度的**。通过一只可爱的柯基犬作为陪伴形象，我们希望让每一次任务完成、每一个灵感捕捉、每一个重要时刻，都变成一次温暖的互动。让忙碌的人也能感受到被陪伴的幸福，让枯燥的任务变得生动有趣。

**产品愿景：让记录变得不再枯燥，让忙碌的人也能感受到温暖。**

### 1.3 目标用户

#### 主要用户群体

| 用户类型           | 特征描述                                    | 核心需求                          | 社会公益价值                             |
| ------------------ | ------------------------------------------- | --------------------------------- | ---------------------------------------- |
| **上班族**   | 25-35岁，工作节奏快，需要管理多项任务和会议 | 高效记录、准时提醒、缓解工作压力  | 缓解职场焦虑，提供情感陪伴，预防 burnout |
| **学生群体** | 18-25岁，学习任务多，需要规划作业和考试     | 课程管理、 deadline提醒、学习动力 | 减轻学业压力，培养自律习惯，提供正向激励 |
| **老年群体** | 55岁以上，需要记录日常事务、用药提醒        | 简单易用、大字体、语音输入        | 缓解孤独感，提供日常陪伴，辅助记忆管理   |

#### 用户身份选择

首次使用APP时，用户可以选择自己的身份类型：

- **上班族模式**：界面偏向简洁高效，问候语侧重职场鼓励
- **学生模式**：界面偏向活泼轻松，问候语侧重学业加油
- **老年模式**：界面简洁大字体，语音输入为主，柯基陪伴更频繁

用户可以随时在设置中切换身份，APP会自动调整界面风格、问候语内容和柯基互动频率。

### 1.3 核心价值主张

#### 三大核心价值

| 价值维度            | 具体体现                             | 用户收益                             |
| ------------------- | ------------------------------------ | ------------------------------------ |
| **🐕 陪伴感** | 柯基犬全程陪伴，从晨间问候到任务完成 | 减少孤独感，增加使用愉悦度           |
| **🎯 高效性** | 快速记录、智能分类、准时提醒         | 提升任务管理、灵感捕捉、日期追踪效率 |
| **💝 温暖感** | 治愈系设计、贴心问候、节日关怀       | 缓解压力，提升幸福感                 |

#### 差异化优势

与市面上冷冰冰的待办APP不同，「刻记+」：

1. **原创陪伴系统** — 不是简单的 mascot，而是有情绪、有反应、会成长的柯基伙伴
2. **身份自适应** — 根据用户身份（上班族/学生）提供定制化的界面和问候
3. **节日节气关怀** — 自动提醒重要日期，并送上专属问候语
4. **完全原创** — 不参考现有产品，打造独特的用户体验

## 2. 用户场景与故事

### 2.1 用户画像

#### 用户A：小林，25岁，职场新人

> "每天被各种会议和任务淹没，常常忘记重要的事情。我想要一个不只是提醒，还能给我一点鼓励的工具。"

- **身份**：上班族
- **痛点**：任务多、压力大、容易焦虑
- **期望**：高效管理 + 情感陪伴

#### 用户B：小雨，20岁，大三学生

> "学业任务繁重，经常忘记作业截止日期。希望有个可爱的东西能陪我一起学习，让枯燥的任务变得有趣。"

- **身份**：学生
- **痛点**：deadline管理、学习动力不足
- **期望**：任务管理 + 正向激励

#### 用户C：王阿姨，68岁，退休教师

> "年纪大了，经常忘记吃药和买菜。儿女都在外地工作，平时一个人在家，有时候挺孤单的。希望有个简单好用的工具帮我记住事情，还能陪我说说话。"

- **身份**：老年群体
- **痛点**：记忆力衰退、用药管理、孤独感
- **期望**：简单易用 + 语音提醒 + 情感陪伴

### 2.2 使用场景

#### 场景一：晨间问候 ☀️

> *侧重功能：晨间问候 + 柯基陪伴*

---

**【小林的故事】**

早上7:30，闹钟响了。我迷迷糊糊地拿起手机，习惯性地打开「刻记+」。

屏幕亮起，一只圆滚滚的柯基正趴在界面上，看到我醒来，它摇了摇尾巴，弹出今天的问候：

> 🐕 **"早安小林！新的一天，柯基陪你一起加油！今天有3个任务等你完成，还有2个灵感等你记录哦~"**

我笑了笑，点开待办列表。柯基在旁边转了个圈，似乎在说"加油"。

这种被等待、被陪伴的感觉，让我对今天的工作没那么抗拒了。

---

#### 场景二：任务完成 ✅

> *侧重功能：任务完成 + 柯基互动*

---

**【小雨的故事】**

下午4点，终于写完了那篇拖了很久的论文。我在APP里勾选了"完成论文初稿"这个任务。

刚勾选完，屏幕上的柯基突然跳了起来，开心地转了三圈，然后送给我一颗小星星：

> 🐕 **"太棒了小雨！你又完成了一项大任务！柯基为你骄傲！⭐"**

看着柯基开心的样子，我突然觉得很有成就感。这种被"看见"、被"肯定"的感觉，比单纯划掉一个任务要温暖得多。

我顺手摸了摸屏幕上的柯基（虽然它不会真的感觉到），它眯起眼睛，尾巴摇得更欢了。

#### 场景三：节日关怀 🎉

> *侧重功能：节日提醒 + 专属问候*

今天是冬至。说实话，我差点忘了这个节气——毕竟一个人在外打拼，传统节日往往被忽略。

但早上打开APP时，柯基穿着小小的围巾，界面变成了暖色调，弹出一条消息：

我愣了一下，然后笑了。原来还有人（虽然是只柯基）记得提醒我过节。

我回复了一句"谢谢柯基"，它开心地摇了摇尾巴，界面上飘起了小小的雪花动画。

那一刻，我觉得这个冬天没那么冷了。

---

#### 场景四：用药提醒 💊

> *侧重功能：语音提醒 + 大字体界面 + 柯基陪伴*

下午3点，手机响了。我戴上老花镜，点开「刻记+」。

屏幕上，柯基穿着小围裙，旁边显示着大大的字：

> 🐕 **"王阿姨，该吃降压药啦！记得配温水哦~ 柯基提醒您：健康最重要！💊"**

我笑着点了"已完成"，柯基开心地摇了摇尾巴，送给我一颗小爱心：

> 🐕 **"真棒！王阿姨最听话了~ 明天柯基还会提醒您的！"**

我顺手用语音记录了今天的感受："今天血压正常，心情也不错。"柯基在旁边认真听着，尾巴轻轻摇晃，好像在说"我在听呢"。

## 3. 功能需求

> **优先级说明**：
>
> - 🔴 **M (Must have)**：必须有，产品核心功能
> - 🟡 **S (Should have)**：应该有，对体验影响大
> - 🟢 **C (Could have)**：可以有，锦上添花

### 3.1 三大核心记录功能

「刻记+」不再只是单纯的待办管理工具，而是集**待办管理**、**灵感记录**、**特殊日期记录**于一体的三位一体生活记录系统。三大功能相互独立又彼此关联，满足用户在不同场景下的记录需求。

---

### 3.1 待办管理功能

#### 3.1.1 创建待办事项 🔴 M

**功能描述**：用户可以创建新的待办事项

**输入**：

- 待办标题（必填，最多100字）
- 待办详情（可选，最多500字）
- 分类标签（可选，默认"未分类"）
- 截止日期和时间（可选）
- 提醒时间（可选）
- 优先级（可选：高/中/低，默认"中"）
- 是否重复（可选：不重复/每天/每周/每月）

**输出**：

- 待办事项创建成功，显示在列表中
- 柯基显示"已记录"动画反馈

**交互流程**：

1. 用户点击"+"按钮或语音按钮
2. 进入创建页面，输入信息
3. 点击保存，返回列表页
4. 柯基播放"记录成功"动画

#### 3.1.2 编辑待办事项 🔴 M

**功能描述**：用户可以修改已创建的待办

**输入**：待办的所有字段均可修改

**输出**：更新后的待办信息

**交互流程**：

1. 用户长按或点击编辑按钮
2. 进入编辑页面
3. 修改内容后保存

#### 3.1.3 完成任务 🔴 M

**功能描述**：用户标记待办为已完成

**输入**：点击待办项前的复选框

**输出**：

- 待办项显示为完成状态（划线、变淡）
- 柯基播放庆祝动画（跳跃+撒花）
- 显示完成鼓励语
- 完成任务数+1

**交互流程**：

1. 用户点击复选框
2. 待办项动画过渡到完成状态
3. 柯基播放庆祝动画（2-3秒）
4. 显示浮动文字"太棒了！又完成一个！"

#### 3.1.4 删除待办 🔴 M

**功能描述**：删除不需要的待办

**输入**：左滑待办项或进入编辑选择删除

**输出**：待办从列表中移除

**特殊说明**：

- 已完成的待办自动归档，保留30天后可清理
- 删除前显示确认对话框

#### 3.1.5 待办列表展示 🔴 M

**功能描述**：展示所有待办事项

**展示维度**：

- 按时间：今天/明天/未来/已完成
- 按优先级：高优先级置顶
- 按分类：工作/学习/生活/其他

**列表项显示**：

- 标题（单行，过长截断）
- 截止时间（如有）
- 优先级标识（颜色区分）
- 分类标签
- 重复标识（如有）

#### 3.1.6 子任务拆分 🟡 S

**功能描述**：复杂任务可以拆分子步骤

**输入**：

- 父任务下添加子任务（最多10个）
- 子任务标题（最多50字）

**输出**：

- 父任务显示进度（如"2/5完成"）
- 子任务可单独勾选完成

**交互流程**：

1. 在创建/编辑页开启"拆分子任务"
2. 添加子任务列表
3. 主列表显示进度条

#### 3.1.7 快速添加 🟡 S

**功能描述**：从系统快捷方式快速创建待办

**入口**：

- 通知栏快捷按钮
- 桌面小部件
- 音量键快捷（可选配置）

**输入**：仅标题（快速记录）

**输出**：创建成功提示

#### 3.1.8 智能分类 🟢 C

**功能描述**：根据关键词自动推荐分类

**规则示例**：

- 含"会议/报告/PPT"→ 工作
- 含"作业/考试/论文"→ 学习
- 含"买/超市/快递"→ 生活

**输入**：待办标题

**输出**：自动选中推荐分类（用户可修改）

### 3.2 灵感记录功能

灵感记录是「刻记+」三大核心功能之一，专为捕捉用户随时涌现的创意、想法和感悟而设计。与待办管理不同，灵感记录更注重自由表达和长期积累，是用户个人知识库和创意源泉。

#### 3.2.1 功能定位与价值

**功能定位**：

- **学习场景**：记录课堂笔记、读书笔记、学习心得、知识点总结
- **工作场景**：记录项目灵感、会议要点、创意构思、问题解决方案
- **生活场景**：记录旅行计划、美食发现、生活感悟、随手拍配文

**核心价值**：

| 价值维度             | 具体体现                                     | 用户收益               |
| -------------------- | -------------------------------------------- | ---------------------- |
| **即时捕捉**   | 随时随地快速记录灵感，防止好想法流失         | 不错过任何创意火花     |
| **系统整理**   | 通过标签和分类管理，将零散灵感组织成知识体系 | 从碎片到体系的知识沉淀 |
| **时间线回顾** | 按时间线展示灵感演进，见证思维成长轨迹       | 看见自己的成长与变化   |
| **跨功能关联** | 灵感可关联待办和特殊日期，想法转化为行动     | 从想到做的无缝衔接     |

#### 3.2.2 创建灵感记录 🔴 M

**功能描述**：用户可以快速创建灵感记录

**输入**：

- 灵感标题（必填，最多100字）
- 灵感内容（必填，支持富文本：加粗/斜体/列表/引用）
- 标签（可选，支持多标签，如#产品 #设计 #读书）
- 关联图片（可选，无数量限制）
- 关联卡片（可选，通过@提及关联待办或特殊日期）

**输出**：

- 灵感记录创建成功，按时间线展示
- 柯基显示"灵感已捕捉"动画反馈

**交互流程**：

1. 用户点击底部导航栏中央⊕按钮 → 选择"💡灵感"气泡
2. 进入灵感编辑页，输入标题和内容
3. 可添加标签（#开头，自动识别）
4. 可插入图片（相册/相机）
5. 可通过@提及关联其他卡片
6. 点击保存，返回灵感列表页
7. 柯基播放"灵感捕捉成功"动画

#### 3.2.3 灵感列表展示 🔴 M

**功能描述**：以时间线形式展示所有灵感记录

**展示维度**：

- **时间线分组**：按日期分组（今天/昨天/本周/更早）
- **标签筛选**：点击标签快速筛选相关灵感
- **搜索**：支持标题和内容关键词搜索

**灵感卡片显示**：

- 标题（单行，过长截断）
- 内容预览（最多2行，超出省略号）
- 缩略图（最多显示2张，多余显示"+N"）
- 标签（主色浅背景圆角标签）
- 创建时间（12sp灰色）
- 关联提示（如有，显示"🔗 关联: @类型:标题"）

#### 3.2.4 灵感详情与编辑 🟡 S

**功能描述**：查看灵感完整内容，支持编辑和删除

**详情页展示**：

- 完整标题和内容（富文本渲染）
- 所有关联图片（可点击查看大图）
- 所有标签（可点击筛选同标签灵感）
- 关联卡片列表（点击可跳转）

**编辑功能**：

- 修改标题、内容、标签
- 添加/删除图片
- 添加/移除关联卡片

#### 3.2.5 灵感空状态设计 🟡 S

**空状态触发**：当用户首次进入灵感页面或删除所有灵感时

**空状态内容**：

- 柯基动画：灯泡图标闪烁（💡）
- 文案："还没有灵感记录~"
- 副文案："点击下方按钮记录你的第一个灵感吧！"
- CTA按钮："💡 记录灵感"

### 3.3 特殊日期记录功能

特殊日期记录是「刻记+」三大核心功能之一，专注于帮助用户记录和管理生命中的重要时刻。无论是期待已久的生日、紧张的考试日期，还是纪念特别的日子，都能以直观的天数卡片形式呈现。

#### 3.3.1 功能定位与价值

**功能定位**：

- **期待管理**：倒计时显示距离重要事件的天数，增加期待感
- **纪念管理**：正计时显示从特别日子开始已经过去的天数，珍藏回忆
- **生活仪式**：帮助用户建立和维持生活中的仪式感

**核心价值**：

| 价值维度             | 具体体现                             | 用户收益                   |
| -------------------- | ------------------------------------ | -------------------------- |
| **时间可视化** | 将抽象的日期转化为直观的天数         | 对时间流逝有具体感知       |
| **情感连接**   | 通过天数变化增强对重要日子的情感投入 | 加深与重要人事物的情感纽带 |
| **生活仪式**   | 记录生日、纪念日、节日等，培养仪式感 | 让生活更有温度和意义       |
| **计划辅助**   | 倒计时提醒帮助用户提前准备           | 从容应对重要事件           |

#### 3.3.2 创建特殊日期 🔴 M

**功能描述**：用户创建特殊日期记录

**输入**：

- 标题（必填，如"生日"、"期末考试"、"在一起纪念日"）
- 目标日期（必填，精确到天）
- 计时模式（必填）：
  - **倒计时**：距离目标日期还有多少天（期待未来）
  - **正计时**：从目标日期起已经多少天（纪念过去）
- 备注（可选，最多200字）
- 关联图片（可选，最多1张封面图）
- 关联卡片（可选，通过@提及关联待办或灵感）

**输出**：

- 特殊日期创建成功，显示在列表中
- 根据计时模式显示"还有X天"或"已经X天"

**交互流程**：

1. 用户点击底部导航栏中央⊕按钮 → 选择"📅日期"气泡
2. 进入日期编辑页，输入标题和选择目标日期
3. 选择计时模式（倒计时/正计时）
4. 可添加备注、图片和关联卡片
5. 点击保存，返回日期列表页

**天数颜色规则**（遵循设计规范12.1.2.3功能色）：

| 状态               | 天数颜色           | 设计意图               |

## 4. 柯基陪伴系统

> 柯基陪伴系统是「刻记+」的核心差异化特色，也是产品**社会公益属性**的重要体现。它不仅是一个虚拟伙伴，更是一个专注于**习惯培养、心灵陪伴、心理疗愈**的情感支持系统。

### 4.0 社会公益价值定位

柯基陪伴系统承载着产品的社会公益使命，致力于解决现代社会中日益突出的**心理健康**和**情感孤独**问题。

#### 4.0.1 社会背景与问题

| 社会问题           | 影响人群 | 具体表现                                       |
| ------------------ | -------- | ---------------------------------------------- |
| **职场焦虑** | 上班族   | 工作压力大、burnout风险高、缺乏情感宣泄渠道    |
| **学业压力** | 学生群体 | 考试焦虑、学习动力不足、自我认同感低           |
| **老年孤独** | 老年群体 | 空巢老人增多、社交圈缩小、记忆力衰退           |
| **数字孤独** | 全年龄段 | 社交媒体替代真实互动、虚拟关系无法满足情感需求 |

#### 4.0.2 柯基陪伴系统的公益价值

| 公益维度           | 目标人群                 | 解决方式                                           | 预期效果                                 |
| ------------------ | ------------------------ | -------------------------------------------------- | ---------------------------------------- |
| **习惯培养** | 学生、上班族             | 通过任务完成激励、连续打卡奖励、柯基成长反馈       | 帮助用户建立自律习惯，提升自我效能感     |
| **心灵陪伴** | 全年龄段，特别是独居人群 | 24小时在线的柯基伙伴、晨间问候、睡前晚安、节日关怀 | 缓解孤独感，提供情感寄托，增强生活仪式感 |
| **心理疗愈** | 高压人群、焦虑人群       | 正向反馈机制、成就系统、情绪识别与安抚、治愈系动画 | 减轻焦虑情绪，提升幸福感，预防轻度抑郁   |

#### 4.0.3 三大核心公益方向

**1. 习惯培养——从自律到自信**

柯基陪伴系统通过游戏化的任务完成机制，帮助用户逐步建立良好的生活和学习习惯：

- **任务完成激励**：每完成一个任务，柯基都会给予正向反馈（动画+鼓励语），让完成任务变成一件有成就感的事
- **连续打卡奖励**：连续完成任务会触发柯基的特殊庆祝动画，强化正向行为
- **习惯养成可视化**：通过数据统计和成就系统，让用户直观看到自己的进步
- **适用人群**：学生（学习习惯）、上班族（工作效率习惯）、老年人（健康生活 habit）

**2. 心灵陪伴——从孤独到温暖**

柯基陪伴系统提供全天候的情感陪伴，特别关注独居人群和老年群体：

- **晨间问候**：每天首次打开APP，柯基会用温暖的问候开启用户的一天
- **睡前晚安**：晚上使用APP时，柯基会提醒用户早点休息
- **节日关怀**：传统节日和节气自动送上专属问候，让用户感受到被记得
- **空闲陪伴**：当用户长时间未使用时，柯基会发送"想念你"的推送
- **适用人群**：独居上班族、空巢老人、异地求学的学生

**3. 心理疗愈——从焦虑到平静**

柯基陪伴系统通过治愈系设计和正向心理学原理，帮助用户缓解压力：

- **情绪识别**：根据用户任务完成情况和使用习惯，识别用户情绪状态
- **情绪安抚**：当检测到用户压力大时（如待办堆积），柯基会显示担心表情并送上安慰
- **正念引导**：通过呼吸动画、冥想引导等方式帮助用户放松
- **成就肯定**：通过成就系统记录用户的每一个小进步，增强自我认同
- **适用人群**：高压职场人群、考试焦虑学生、情绪敏感人群

#### 4.0.4 特殊群体关怀设计

**老年群体专属设计**：

- **大字体模式**：所有文字放大至18sp以上
- **语音交互**：支持语音输入和语音播报，减少打字负担
- **简化界面**：隐藏复杂功能，保留核心待办和日期记录
- **用药提醒**：支持设置用药时间和剂量提醒
- **紧急联系**：一键拨打紧急联系人电话
- **柯基陪伴增强**：柯基互动更频繁，陪伴感更强

**学生群体专属设计**：

- **学习专注模式**：开启后柯基显示"学习中"状态，屏蔽娱乐通知
- **考试倒计时**：自动识别考试日期，提供备考建议
- **成绩记录**：支持记录考试成绩，追踪学习进步
- **励志语录库**：根据学习阶段推送不同的励志语录

**上班族专属设计**：

- **工作番茄钟**：25分钟专注+5分钟休息，柯基陪伴倒计时
- **会议提醒**：提前15分钟提醒，附带会议准备清单
- **下班仪式**：完成当日所有任务后，柯基播放"下班庆祝"动画
- **压力释放**：长按柯基触发"捏捏"动画，缓解工作压力

### 4.3 情绪系统

#### 4.3.1 情绪状态定义

**情绪影响因素**：

- 用户任务完成情况
- 用户活跃度（使用频率）
- 特殊时间节点（节日、深夜等）

**情绪状态列表**：

| 情绪状态       | 触发条件           | 柯基表现           | 问候语风格 |
| -------------- | ------------------ | ------------------ | ---------- |
| **开心** | 今日任务完成≥80%  | 蹦跳、摇尾巴、笑眼 | 热情、鼓励 |
| **普通** | 正常状态           | 趴卧、普通表情     | 平和、友好 |
| **期待** | 有未完成任务       | 歪头、闪亮眼睛     | 提醒、期待 |
| **担心** | 任务即将超期       | 眉毛下垂、不安     | 关切、提醒 |
| **困倦** | 深夜（23:00-6:00） | 打哈欠、半闭眼     | 温柔、关心 |
| **兴奋** | 连续完成多个任务   | 奔跑、跳跃         | 激动、赞美 |
| **失落** | 连续3天未完成任务  | 耳朵下垂、低头     | 鼓励、安慰 |

#### 4.3.2 情绪变化规则

**情绪计算公式**（简化版）：

```
情绪值 = 基础值(50) 
       + 今日完成率 × 30
       + 连续活跃天数 × 5
       - 超期任务数 × 10
       + 特殊事件加成
```

**情绪值对应状态**：

| 情绪值范围 | 情绪状态  |
| ---------- | --------- |
| 80-100     | 开心/兴奋 |
| 60-79      | 期待      |
| 40-59      | 普通      |
| 20-39      | 担心      |
| 0-19       | 失落      |

#### 4.3.3 情绪恢复机制

**自动恢复**：

- 每天凌晨重置为基础值50
- 用户完成任务后情绪值上升
- 用户互动后情绪值小幅上升

**用户干预**：

- 点击柯基互动：+2情绪值
- 完成一个任务：+5情绪值
- 连续完成任务：额外+3情绪值

## 5. 界面设计规范

> 设计理念：**治愈、温暖、舒适**——圆润柔和、低饱和度配色、充足留白，让用户感到放松。

### 5.1 设计风格

**治愈系设计关键词**：圆润（大圆角16dp+）、柔和（低饱和度）、留白（充足间距）、层次（轻微阴影）、呼吸感（柔和动画）

**设计原则**：温暖优先、减少焦虑（避免红色警告）、正向反馈、全局一致。

### 5.2 主题配色

提供6种预设主题，用户可自由切换：

| 主题        | 主色        | 情感联想           |
| ----------- | ----------- | ------------------ |
| 暖阳橙 🧡   | `#FF9A5C` | 温暖、活力（默认） |
| 樱花粉 🌸   | `#FFB5C2` | 柔和、浪漫         |
| 薄荷绿 🌿   | `#7EC8A0` | 清新、宁静         |
| 天空蓝 ☁️ | `#7EB8DA` | 治愈、平静         |
| 薰衣紫 💜   | `#B8A0D4` | 优雅、温柔         |
| 奶茶棕 🍵   | `#C4A882` | 温馨、安心         |

深色模式下各主题色自动降低亮度30%。支持跟随系统、手动切换、定时切换三种模式。

### 5.3 身份差异化设计（创意点）

APP根据用户身份自动调整界面风格，这是本产品的**核心创意点之一**：

| 对比维度     | 上班族模式          | 学生模式                 |
| ------------ | ------------------- | ------------------------ |
| 默认主题     | 奶茶棕/天空蓝       | 暖阳橙/樱花粉            |
| 页面布局     | 紧凑高效            | 宽松活泼                 |
| 图标风格     | 线性简洁            | 圆润可爱                 |
| 默认分类     | 工作/会议/报告/个人 | 学习/作业/考试/社团/生活 |
| 问候语风格   | 简洁有力            | 活泼有趣                 |
| 柯基默认姿态 | 坐立（认真）        | 趴卧（悠闲）             |

**问候语示例**：

- 上班族："早安！今天有3个任务，最紧急的是「提交周报」，加油搞定它！"
- 学生："早安呀！📚 今天有2个作业要交，「高数作业」明天就截止啦，要抓紧哦~"

### 5.4 深色模式特殊处理（创意点）

- **柯基形象**：保持原样，周围添加微弱暖色光晕，营造"夜灯陪伴"的氛围
- **卡片阴影**：减弱阴影强度，改用微妙的边框，避免深色背景上的阴影突兀
- **图片/插画**：适当降低亮度，与深色背景协调

### 5.5 动效规范

| 场景       | 时长  | 风格                  |
| ---------- | ----- | --------------------- |
| 页面切换   | 300ms | 滑动，Ease-in-out     |
| 卡片展开   | 250ms | 高度变化，Ease-out    |
| 复选框勾选 | 200ms | 缩放+颜色，Ease-out   |
| 弹窗出现   | 300ms | 从底部滑入，Ease-out  |
| 按钮点击   | 100ms | 缩放0.95，Ease-in-out |

柯基动画分三层：前景（主体动画）、中景（星星/爱心特效）、背景（氛围颜色渐变）。
