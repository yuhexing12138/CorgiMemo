# 待办卡片拖拽排序修复验证流程

## 核心验证命令

### 1. 实时监控（推荐）

```powershell
# 清空旧日志
adb logcat -c

# 实时过滤输出
adb logcat CorgiMemo-Reorder:D *:S
```

### 2. 测试完成后一次性导出

```powershell
# dump 全部日志到文件
adb logcat -d -s CorgiMemo-Reorder:D > C:\Users\EDY\Desktop\CorgiMemo\reorder_log.txt

# 查看日志
Get-Content reorder_log.txt
```

### 3. 过滤关键信息

```powershell
# 仅看批量更新
adb logcat -d -s CorgiMemo-Reorder:D | Select-String "updateTodos"

# 统计调用次数
adb logcat -d -s CorgiMemo-Reorder:D | Select-String "updateTodos" | Measure-Object
```
